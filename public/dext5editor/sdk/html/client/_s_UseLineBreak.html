DEXT5 Editor :: UseLineBreak

XML : <use_line_break>.....</use_line_break>, Config : config.UseLineBreak
XML : <use_line_break word_break_type="....."></use_line_break>, Config : config.WordBreakType
XML : <use_line_break word_wrap_type="....."></use_line_break>, Config : config.WordWrapType

Config 설정이 우선 적용됩니다.
줄 바꿈 기능 사용 여부 및 줄 바꿈 기준, 가로길이를 설정합니다.

remarks

use_line_break  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 줄 바꿈 기능을 사용합니다.
word_break_type  기본값은 "0" 이며 기본값, "1" 로 설정 시 글자단위, "2" 로 설정 시 단어단위로 줄바꿈 합니다.
word_wrap_type  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 글 내용의 가로길이를 맞추도록 설정합니다.

sample code

줄 바꿈 기능을 단어단위로 글 내용의 가로길이를 맞춰서 하도록 설정합니다.

XML

.....
<use_line_break word_break_type="2" word_wrap_type="1">1</use_line_break>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
