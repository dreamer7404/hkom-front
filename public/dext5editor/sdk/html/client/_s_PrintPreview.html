DEXT5 Editor :: PrintPreview

XML : <print_preview>.....</print_preview>, Config : config.PrintPreview

Config 설정이 우선 적용됩니다.
인쇄 미리보기 기능 사용여부를 설정합니다.

remarks

기본값은 "0" 사용 안함, "1" 로 설정 시 인쇄 미리보기 기능을 사용합니다.

sample code

인쇄 미리보기 기능을 사용으로 설정합니다.

XML

.....
<dext5>
    <license>
        .....
        <print_preview>1</print_preview>
        .....
    </license>
    .....
</dext5>

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        
        DEXT5.config.PrintPreview = "1";

        new Dext5editor("editor1");

    </script>       
</div>
