DEXT5 Editor :: MoveStyleTagToHead

XML : <move_style_tag_to_head>.....</move_style_tag_to_head>, Config : config.MoveStyleTagToHead

Config 설정이 우선 적용됩니다.
set API 사용시 body 안에 있는 style태그를 작성영역의 head 태그 안으로 위치하도록 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 set API 사용할 경우 body 안에 있는 style태그를 작성영역의 head 태그 안으로 위치하도록 설정합니다.

sample code

set API 사용시 body 안에 있는 style태그를 작성영역의 head 태그 안으로 위치하도록 설정합니다.

XML

.....
<setting>
    <move_style_tag_to_head>1</move_style_tag_to_head>
    .....
</setting>
.....
