DEXT5 Editor :: StatusBar

XML : <status_bar>.....</status_bar>, Config : config.StatusBarItem
XML : <status title=""></status_bar>, Config : config.StatusBarItemTitle

Config 설정이 우선 적용됩니다.
에디터의 보기화면(statusBar) 메뉴를 원하는 탭으로 표시하거나 숨기는 기능을 설정합니다.
각 탭의 title 설정이 가능합니다.

remarks

상태바의 메뉴
메뉴명 command 명
디자인 design
HTML source
미리보기 preview
TEXT text

title  해당 탭(li태그)의 title을 지정합니다. (마우스를 올려서도 확인이 가능합니다.)

sample code

에디터 보기화면(statusBar) 메뉴를 디자인, HTML, 미리보기로 설정합니다.
각 탭의 title을 각각 "작성", "HTML소스", "미리보기"로 설정 합니다.

XML

.....
<status_bar>
    <status title="작성">design</status>
    <status title="HTML소스">source</status>
    <status title="미리보기">preview</status>
</status_bar>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.StatusBarItem = "design,source,preview";
        DEXT5.config.StatusBarItemTitle = "작성,HTML소스,미리보기"

        new Dext5editor("editor1");

    </script>       
</div>
