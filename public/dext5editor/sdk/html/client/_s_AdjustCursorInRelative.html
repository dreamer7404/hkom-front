[IE 브라우저 전용]

DEXT5 Editor :: AdjustCursorInRelative

XML : <adjust_cursor_in_relative>.....</adjust_cursor_in_relative>, Config : config.AdjustCursorInRelative

Config 설정이 우선 적용됩니다.
IE 브라우저에서 span 태그에 position: relative 스타일이 적용되어 있을경우 마우스 클릭 시 커서 이동이 되는 현상을 보정 하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 IE 브라우저에서 span 태그에 position: relative 스타일이 적용되어 있을경우 마우스 클릭 시 커서 이동이 되는 현상을 보정 합니다.

sample code

IE 브라우저에서 span 태그에 position: relative 스타일이 적용되어 있을경우 마우스 클릭 시 커서 이동이 되는 현상을 보정 합니다.

XML

.....
<setting>
    <adjust_cursor_in_relative>1</adjust_cursor_in_relative>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AdjustCursorInRelative = "1";

        new Dext5editor("editor1");

    </script>       
</div>
