DEXT5 Editor :: SetDefaultTagInEmptyCell

XML : <set_default_tag_in_empty_cell>.....</set_default_tag_in_empty_cell>,
Config : config.SetDefaultTagInEmptyCell

Config 설정이 우선 적용됩니다.
셀을 마우스 클릭 또는 셀로 키보드 커서 이동 시 빈 셀에 p태그를 삽입합니다.

remarks

기본값은 "0"으로 셀에 마우스클릭 및 키보드 커서 이동 시 P태그 삽입 안함,
"1"로 설정 시 셀에 마우스클릭 및 키보드 커서 이동 시 P태그를 삽입합니다.

sample code

셀을 마우스 클릭 또는 셀로 키보드 커서 이동 시 빈 셀에 p태그를 삽입합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <set_default_tag_in_empty_cell>1</set_default_tag_in_empty_cell>
        .....
    </table_property>
    .....
</setting>
.....
