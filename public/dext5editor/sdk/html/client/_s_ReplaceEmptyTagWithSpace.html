DEXT5 Editor :: ReplaceEmptyTagWithSpace

XML : <replace_empty_tag_with_space>.....</replace_empty_tag_with_space>,
Config : config.ReplaceEmptyTagWithSpace

Config 설정이 우선 적용됩니다.
get API 사용시 빈 태그를 &nbsp;로 치환하도록 설정합니다.

remarks

기본값은 "1"로 get API 사용시 빈 태그를 &nbsp;로 치환함,
"0"으로 설정 시 치환하지 않습니다.
