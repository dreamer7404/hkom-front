DEXT5 Editor :: ReplaceRgbToHex

XML : <replace_rgb_to_hex>.....</replace_rgb_to_hex>, Config : config.ReplaceRgbToHex

Config 설정이 우선 적용됩니다.
get API 사용시 style의 rgb 값을 hex 값으로 변경하도록 설정합니다.

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 get API 사용시 style의 rgb 값을 hex 값으로 변경합니다.

sample code

get API 사용시 style의 rgb 값을 hex 값으로 변경합니다.

XML

.....
<setting>
    <replace_rgb_to_hex>1</replace_rgb_to_hex>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ReplaceRgbToHex = "1";

        new Dext5editor("editor1");

    </script>       
</div>
