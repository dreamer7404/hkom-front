DEXT5 Editor :: DisableTabletap

XML : <disable_tabletap>.....</disable_tabletap>, Config : config.DisableTabletap

Config 설정이 우선 적용됩니다.
표에서 tab 입력시 셀 이동 제한여부를 설정합니다.

remarks

기본값은 "0" 이며 표에서 tab 입력시 셀 이동 함,
"1" 로 설정 시 표에서 tab 입력시 셀 이동 하지않도록 설정합니다.

sample code

표에서 tab 입력시 셀 이동 하지않도록 설정합니다.

XML

.....
<setting>
    <disable_tabletap>1</disable_tabletap>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.DisableTabletap = "1";

        new Dext5editor("editor1");

    </script>
