DEXT5 Editor :: WebFontCssUrl

XML : <web_font_css_url>.....</web_font_css_url>, Config : config.WebFontCssUrl
XML : <web_font_css_url always_set="....."></web_font_css_url>, Config : config.WebFontCssAlwaysSet

Config 설정이 우선 적용됩니다.
에디터 로드시 사용자의 web font CSS를 에디터 body의 head부분에 import 해주도록 설정합니다.

remarks

에디터 작성영역 및 글꼴메뉴(콤보박스)에 적용됩니다.
web_font_css_url  import할 사용자의 CSS url(http://를 포함한 전체)을 입력합니다.
always_set  해당 CSS를 항상 포함할지의 여부를 설정합니다. (0 : 최초로드시에만 포함, 1 : 항상포함)

sample code

에디터 로드시 사용자의 web font CSS를 에디터 body의 head부분에 import후 항상 포함해주도록 설정합니다.

XML

.....
<setting>
    <web_font_css_url always_set="1">.....</web_font_css_url>
    .....
</setting>
.....
