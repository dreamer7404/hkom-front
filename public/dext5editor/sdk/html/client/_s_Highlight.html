DEXT5 Editor :: Highlight

XML : <highlight>.....</highlight>, Config : config.Highlight.Use
XML : <highlight color="....."></highlight>, Config : config.Highlight.Color

Config 설정이 우선 적용됩니다.
검색 하이라이팅(setHighlight API) 기능 사용여부를 설정합니다.
검색 하이라이팅 시 적용할 하이라이트 색을 설정합니다.

remarks

setHighlight API를 사용하기위한 설정입니다.
highlight  기본값은 "0" 이고 사용 안함, "1"로 설정 시 검색 하이라이팅 기능을 사용합니다.
highlight@color  기본값은 "" 이고, 색상 입력 시 검색 하이라이팅의 색을 설정합니다.

sample code

검색 하이라이팅(setHighlight API) 기능을 사용합니다.
하이라이팅 색상을 rgb(50,100,140)으로 설정합니다.

XML

.....
<setting>
    .....
    <browser_bugfixed>
        <browser_bugfixed delete_table_align="1" replace_align_margin="1" replace_br="1">1</browser_bugfixed>
    </browser_bugfixed>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.Highlight.Use = "1";
        DEXT5.config.Highlight.Color = "rgb(50,100,140)";

        new Dext5editor("editor1");

    </script>       
</div>
