DEXT5 Editor :: insertInput

void insertInput(attributesObj, inputObj, editor_id)

에디터 선택영역에 attributesObj의 정보로 input 객체를 생성하여 삽입합니다.
inputObj 오브젝트가 에디터의 body에 존재하면 해당 input 객체를 attributesObj의 정보로 변경합니다.

return value

없음.

parameters

attributesObj  생성 또는 교체할 input 객체의 정보를 의미합니다.
inputObj  에디터 body에 존재하는 교체대상의 input 객체를 의미합니다. (생성을 원하면 null 입력)
editor_id  에디터의 id를 의미합니다.

remarks

attributesObj는 다음과 같은 정보를 설정할 수 있습니다.
type, name, id, class, title, value, size, maxlength, disabled, readonly,
checked, alt, style.width, style.height, style.text_align

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Input(){
        // 테스트용 input 생성정보
        var Obj = {
            type:text, 
            name:"DEXT5", 
            id:"DEXT5EDITOR", 
            class:"DEXT5CLASS", 
            title:"DEXT5TITLE",
            value:"DEXT5VALUE", 
            size:20, 
            maxlength:10,
            disabled:false, 
            readonly:false, 
            //checked:false, radio type 에서만 사용
            alt:"dext5",
            "style.width":"200px", 
            "style.height":"100px", 
            "style.text_align":"center"
        };
        // id가 'editor1'인 Obj정보의 input 객체를 새로 생성하여 삽입합니다.
        DEXT5.insertInput(Obj, null, 'editor1');
        // work something ...
    }

</script>
