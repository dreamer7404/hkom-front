DEXT5 Editor :: ReplaceMsoStyle

XML : <replace_mso_style>.....</replace_mso_style>, Config : config.ReplaceMsoStyle

Config 설정이 우선 적용됩니다.
에디터에서 엑셀내용 복사시 Mso Style 삭제하지 않고 Mso -> dext로 치환하여 속성값으로 추가하도록 설정합니다.

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 에디터에서 엑셀내용 복사시 Mso Style 삭제하지 않고 Mso -> dext로 치환하여 속성값으로 추가합니다.
속성값을 콤마(,)로 구분하여 입력합니다.

sample code

에디터에서 엑셀내용 복사시 Mso Style 삭제하지 않고 Mso -> dext로 치환하여 속성값으로 추가합니다.

XML

.....
<setting>
    <replace_mso_style>.....</replace_mso_style>
    .....
</setting>
