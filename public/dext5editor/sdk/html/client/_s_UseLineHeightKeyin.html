DEXT5 Editor :: UseLineHeightKeyin

XML : <line_height use_keyin="....."></line_height>, Config : config.UseLineHeightKeyin

Config 설정이 우선 적용됩니다.
tool bar의 줄간격 콤보박스에서 키보드 입력(문자입력) 하여 선택 가능하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 tool bar의 줄간격 콤보박스에서 키보드 입력(문자입력) 하여 줄간격 선택이 가능합니다.

sample code

tool bar의 줄간격 콤보박스에서 키보드 입력(문자입력) 하여 선택 가능하도록 설정합니다.

XML

.....
<line_height use_keyin="1">1</line_height>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UseLineHeightKeyin = "1";

        new Dext5editor("editor1");

    </script>       
</div>
