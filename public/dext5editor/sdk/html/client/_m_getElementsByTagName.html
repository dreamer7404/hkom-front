DEXT5 Editor :: getElementsByTagName

HtmlCollection getElementsByTagName(tagName, editor_id)

에디터의 body 내부의 html 소스 중 파라미터의 태그명와 동일한 태그명을 가진 태그들을 HtmlCollection 형태로 반환해줍니다.

return value

에디터 body의 해당 태그명을 가진 html태그 element들을 포함한 HtmlCollection

parameters

tagName  에디터 body내부 html태그의 태그명을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

없음.
