DEXT5 Editor :: OpenDocument.Word

XML : <word max_page="....."></word>, Config : config.OpenDocument.Word.MaxPage
XML : <word max_size="....."></word>, Config : config.OpenDocument.Word.MaxSize

Config 설정이 우선 적용됩니다.
워드파일의 열기 관련 설정을 적용합니다.

remarks

max_page  파일 열기할 word파일의 최대 page 수를 설정합니다.
max_size  파일 열기할 word파일의 최대 용량을 설정합니다.

sample code

워드파일의 열기 관련 설정을 적용합니다.

XML

.....
<setting>
    .....
    <open_document>
        <word max_page="2", max_size="10000"></word>
    </open_document>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.OpenDocument.Word = {

            MaxSize : "10000",
