DEXT5 Editor :: CheckApplyWordBreakForTable

XML : <check_apply_word_break_for_table>.....</check_apply_word_break_for_table>
Config : config.CheckApplyWordBreakForTable

Config 설정이 우선 적용됩니다.
모든 테이블내에서 줄바꿈을 단어 단위로 적용하도록 설정합니다.

remarks

기본값은 "1" 이고 모든 테이블내에서 줄바꿈을 단어 단위로 적용하도록 설정,
"0" 으로 설정 시 적용 하지 않습니다.

sample code

모든 테이블내에서 줄바꿈을 단어 단위로 적용하도록 설정합니다.

XML

.....
<setting>
    .....
    <check_apply_word_break_for_table>1</check_apply_word_break_for_table>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
