DEXT5 Editor :: RemoveTempFolderDataDay

XML : <remove_temp_folder_data_day>.....</remove_temp_folder_data_day>
Config : config.RemoveTempFolderDataDay

Config 설정이 우선 적용됩니다.
temp폴더의 삭제 주기를 설정합니다.

remarks

숫자만 입력합니다. (단위는 "일" 입니다.)
기본값은 "7" 입니다.

sample code

temp폴더의 삭제 주기를 3일로 설정합니다.

XML

.....
<setting>
    <remove_temp_folder_data_day>3</remove_temp_folder_data_day>
    .....
</setting>
.....
