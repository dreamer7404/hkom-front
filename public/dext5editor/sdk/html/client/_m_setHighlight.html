DEXT5 Editor :: setHighlight

void setHighlight(option, editor_id)

에디터 작성영역의 찾을 text를 찾아서 highlight를 적용 또는 해제합니다.

return value

없음.

parameters

option  highlight를 적용 또는 해제 및 찾을 text를 설정합니다. { searchText : "찾을 단어", addOrRemove : "하이라이트 적용 여부"}
editor_id  에디터의 id를 의미합니다.

remarks

에디터 설정파일(dext5editor/config/dext_editor.xml)의 highlight.color가 1이어야 동작합니다.
option.addOrRemove 값이 "add"이면 적용, "remove"이면 해제합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Set(){
        var option = {
            searchText : "DEXT5",
            addOrRemove : "add"
        };
        // id가 editor1인 에디터 작성영역의 option.searchText를 전부 찾아서 Highlight를 적용합니다.
        DEXT5.setHighlight(option, 'editor1');
        // work something ...
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px;"> 
 
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
