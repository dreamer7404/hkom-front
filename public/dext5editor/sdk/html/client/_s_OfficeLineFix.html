DEXT5 Editor :: OfficeLineFix

XML : <office_linefix>.....</office_linefix>, Config : config.OfficeLineFix

Config 설정이 우선 적용됩니다.
PPT, EXCEL에서 html을 get 할 때 테두리 누락을 방지하도록 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 PPT, EXCEL에서 html을 get 할 때 border관련 style이 none이면 제거하여 테두리 누락을 방지합니다.

sample code

PPT, EXCEL에서 html을 get 할 때 테두리 누락을 방지하도록 설정합니다.

XML

.....
<license>
    <office_linefix>1</office_linefix>
    .....
</license>
.....

Config
