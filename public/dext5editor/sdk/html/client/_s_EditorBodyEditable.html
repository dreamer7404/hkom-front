DEXT5 Editor :: EditorBodyEditable

XML : <editor_body_editable>.....</editor_body_editable>, Config : config.EditorBodyEditable
XML : <editor_body_editable mode="....."></editor_body_editable>, Config : config.EditorBodyEditableMode

Config 설정이 우선 적용됩니다.
디자인 영역에 작성 가능 여부를 설정합니다.
작성 불가시 마우스 이벤트 발생여부를 설정합니다.

remarks

XML:editor_body_editable,Config:EditorBodyEditable  기본값은 "1" 이고 작성 가능, "0"으로 설정 시 작성 불가능으로 설정합니다.
XML:mode,Config:EditorBodyEditableMode  기본값은 "0" 이고 설정 안함, EditorBodyEditable의 설정값이 0일때 2로 설정 시 마우스 이벤트를 발생하지 않도록합니다.

sample code

디자인 영역에 작성 불가능으로 설정합니다.
마우스 이벤트도 발생하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <editor_body_editable mode="2">0</editor_body_editable>
    .....
</setting>
