DEXT5 Editor :: addHtmlToSetValue

void addHtmlToSetValue(html, preOrSub, editor_id)

에디터에 set 하는 API를 사용할때 내용의 앞 또는 뒤에 파라미터의 html 소스를 추가하도록 설정합니다.

return value

없음.

parameters

html  앞 또는 뒤에 추가할 html 소스를 의미합니다.
preOrSub  추가할 위치를 의미합니다. (0 : 앞, 1 : 뒤)
editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        var html = '<p>Hi, DEXT5 Editor!!</p>';
        
        // id가 editor1인 에디터가 set API를 사용하면 입력하는 소스의 앞쪽에 html의 내용을 추가합니다.
        DEXT5.addHtmlToSetValue(html, 0, 'editor1');
    }

</script>

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
  
    <!-- 에디터 생성 -->
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>       

</div>
