DEXT5 Editor :: Figure

XML : <figure use="....."></Figure>, Config : config.Figure.Use
XML : <figure default_caption="....."></Figure>, Config : config.Figure.DefaultCaption
XML : <figure figcaption_style="....."></Figure>, Config : config.Figure.FigcaptionStyle
XML : <figure figure_style="....."></Figure>, Config : config.Figure.FigureStyle

Config 설정이 우선 적용됩니다.
이미지 추가시 이미지 설명에 대한 설정들을 적용합니다.

remarks

XML:figure@use,Config:config.Figure.Use  기본값은 "0" 이고 사용 안함, "1"으로 설정 시 이미지 추가할 때 이미지의 설명넣기 기능을 추가합니다.
XML:figure@default_caption,Config:config.Figure.DefaultCaption  이미지 설명 추가시 기본값을 설정합니다. XML:figure@figcaption_style,Config:config.Figure.FigcaptionStyle  css style을 입력하여 figcaption의 style을 설정합니다. XML:figure@figure_style,Config:config.Figure.FigureStyle  css style을 입력하여 figure tag의 style을 설정합니다.

sample code

이미지 추가시 이미지 설명에 대한 설정들을 적용합니다.

XML

.....
<setting>
    .....
    <figure use="1" default_caption="DEXT5 Editor" figcaption_style="color:blue;font-size:14pt" figure_style="border:solid gray"></figure>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.Figure.Use = "1";
        DEXT5.config.Figure.DefaultCaption = "DEXT5 Editor";
        DEXT5.config.Figure.FigcaptionStyle = "color:blue;font-size:14pt;";
        DEXT5.config.Figure.FigureStyle = "border:solid gray;";

        new Dext5editor("editor1");

    </script>       
</div>
