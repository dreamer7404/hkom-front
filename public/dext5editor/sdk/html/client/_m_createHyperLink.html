DEXT5 Editor :: createHyperLink

void createHyperLink(param, editor_id)

에디터 body 현재 선택영역에 하이퍼링크를 추가합니다.

return value

없음.

parameters

param  삽입할 하이퍼링크의 정보를 설정합니다.
param = {
    text:"문구",
    target:"여는방식",
    url:"접속페이지주소"
}
editor_id  적용할 에디터의 id를 의미합니다.

remarks

없음.
