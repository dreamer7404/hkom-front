[관리자 라이선스 전용]

DEXT5 Editor :: ManagerMode

XML : <manager_mode use="....."></manager_mode>
Config : config.ManagerMode
XML : <table_lock default_table_lock_mode="....."></table_lock>
Config : config.DefaultTableLockMode
XML : <table_lock default_lock_name="....."></table_lock>
Config : config.DefaultLockName
XML : <table_lock default_show_lock_icon_user_mode="....."></table_lock>
Config : config.DefaultShowLockIconUserMode
XML : <table_lock check_lock_name="....."></table_lock>
Config : config.CheckLockName
XML : <event_list>.....</event_list>
Config : config.EventList

Config 설정이 우선 적용됩니다.
관리자모드에 관련된 설정을 적용합니다.

remarks

use  관리자기능(Form 요소 관리 및 Table Locker)활성화 여부를 설정합니다.
default_table_lock_mode  기본값은 "2"이며 Resize 및 편집제한, 값이 "1"인 경우 테이블에 대한 Resize만 제한합니다.
default_lock_name  TABLE/TD/TH에 해당 설정값으로 setAttribute된 테이블 또는 셀을 잠급니다.
default_show_lock_icon_user_mode  테이블, 셀의 잠금 여부를 사용자에게 Show(보이기) 또는 Hidden(감추기) 여부를 설정합니다.
                                            (0 : Hidden, 1 : Show)
check_lock_name  타사 에디터와의 호환성을 위한 값으로 타사의 에디터로 저장된 테이블/셀 잠금 기능을 사용허용합니다.
event_list  Form 요소들에 대하여 기본적으로 추가하고자 하는 이벤트명을 설정합니다. (다이얼로그 항목으로 추가)

sample code

관리자모드에 관련된 설정을 적용합니다.

XML

.....
<setting>
    .....
    <manager_mode use="1">
        <table_lock default_table_lock_mode="2" default_show_lock_icon_user_mode="1" default_lock_name="dext5_lock" check_lock_name=""></table_lock>   
        <event_list>
            <event>onblur</event>
            <event>onchange</event>
            <event>onsubmit</event>
            <event>onkeydown</event>
            <event>onkeyup</event>
            <event>onkeypress</event>
            <event>onmousemove</event>
            <event>onmousedown</event>
            <event>onmouseup</event>
            <event>onclick</event>
            <event>ondblclick</event>
        </event_list>
    </manager_mode>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">
