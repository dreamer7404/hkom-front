DEXT5 Editor :: replaceTextAll

void replaceTextAll(paramObj, editor_id)

에디터 작성영역에서 모든 특정 text를 replace text로 변경 해줍니다.

return value

없음.

parameters

paramObj  replaceText의 option parameter object 입니다.
editor_id  에디터의 id를 의미합니다.

remarks

paramObj는 다음과 같은 정보를 포함합니다.
searchText : 셀렉션 할 단어,
replaceText : 교체 할 단어,
caseSensitive : 대소문자 구분 (true || false),
showAlert : alert 발생 여부 (true || false)

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Replace(){
        // option parameter object 설정
        var paramObj = {
            searchText:"Editor", 
            replaceText:"DEXT5", 
            caseSensitive:false, 
            showAlert:false
        }
        // id가 editor1인 에디터에서 paramObj의 값으로 replaceText API를 수행합니다.
        DEXT5.replaceTextAll(paramObj, 'editor1');
        // work something ...
    }

</script>
