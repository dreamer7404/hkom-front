DEXT5 Editor :: ViewImgBase64Source

XML : <paste_image_base64 view_base64_source="....."></paste_image_base64>
Config : config.ViewImgBase64Source

Config 설정이 우선 적용됩니다.
소스보기 모드에서 이미지 src를 base64 그대로 노출하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 소스보기 모드에서 이미지 src를 base64 그대로 노출합니다.

sample code

소스보기 모드에서 이미지 src를 base64 그대로 노출하도록 설정합니다.

XML

.....
<extra_setting>
    <paste_image_base64 view_base64_source="1">1</paste_image_base64>
    .....
</extra_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ViewImgBase64Source = "1";

        new Dext5editor("editor1");

    </script>       
</div>
