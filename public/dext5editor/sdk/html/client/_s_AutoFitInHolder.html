DEXT5 Editor :: AutoFitInHolder

XML : <auto_fit_in_holder>.....</auto_fit_in_holder>, Config : config.AutoFitInHolder

Config 설정이 우선 적용됩니다.
특정 태그(table, img)들을 영역의 크기에 맞게 사이즈를 조정해 주도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 으로 설정 시 table, img 태그를 영역의 크기에 맞게 사이즈를 조정합니다.

sample code

table, img 태그를 영역의 크기에 맞게 사이즈를 조정합니다.

XML

.....
<setting>
    <auto_fit_in_holder>1</auto_fit_in_holder>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AutoFitInHolder = "1";

        new Dext5editor("editor1");

    </script>
