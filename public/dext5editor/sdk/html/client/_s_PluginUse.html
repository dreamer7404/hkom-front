DEXT5 Editor :: PluginUse

XML : <plugin_use>.....</plugin_use>, Config : config.PluginUse

Config 설정이 우선 적용됩니다.
runtime을 ieplugin모드로 설정합니다.

remarks

기본값은 "0" 이며 Runtime의 설정값으로 설정됩니다,
"1" 로 설정 시 runtime을 ieplugin모드로 설정합니다.

sample code

runtime을 ieplugin모드로 설정합니다.

XML

.....
<license>
    <plugin_use>1</plugin_use>
    .....
</license>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PluginUse = "1";

        new Dext5editor("editor1");

    </script>       
</div>
