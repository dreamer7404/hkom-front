DEXT5 Editor :: AllowVideoFileType

XML : <allow_video_file_type>.....</allow_video_file_type>, Config : config.AllowVideoFileType
XML : <allow_video_file_type max_file_size="....."></allow_video_file_type>, Config : config.MaxVideoFileSize

Config 설정이 우선 적용됩니다.
비디오 삽입 시 허용할 확장자를 설정합니다.
비디오 삽입 시 허용할 최대 파일 사이즈를 제한합니다.

remarks

allow_video_file_type   확장자를 ","(콤마)로 구분하여 입력합니다.
max_file_size   용량을 바이트단위까지 입력합니다.

sample code

비디오 삽입 시 허용할 확장자를 "mp4,ogv,webm"로 설정합니다.
비디오 삽입 시 허용할 최대 파일 사이즈를 "30MB"로 설정합니다.

XML
.....
<uploader_setting>
    .....
    <allow_video_file_type max_file_size="30MB">mp4,ogv,webm</allow_video_file_type>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
