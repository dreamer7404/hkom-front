DEXT5 Editor :: XssCheckAttribute

XML : <xss_use check_attribute="....."></xss_use>, Config : config.XssCheckAttribute

Config 설정이 우선 적용됩니다.
에디터의 XSS 방지시 체크할 속성 항목들을 설정합니다.

remarks

XSS 방지시 체크할 속성 항목들을 콤마(,)로 구분하여 입력합니다.

sample code

에디터의 XSS 방지시 "src,background,url,href" 속성들을 체크하도록 설정합니다.

XML

.....
<xss_protection>
    <xss_use check_attribute="src,background,url,href">.....</xss_use>
    .....
</xss_protection>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.XssCheckAttribute = "src,background,url,href";

        new Dext5editor("editor1");

    </script>       
</div>
