DEXT5 Editor :: Security

XML : <security><encrypt_param>.....</encrypt_param></security> Config : config.Security.EncryptParam
XML : <security><file_extension_detector>.....</file_extension_detector></security>
Config : config.Security.FileExtensionDetector

Config 설정이 우선 적용됩니다.
에디터에서 파일을 업로드 할 경우 암호화, 파일변조와 같은 보안에 관련된 설정입니다.

remarks

encrypt_param  기본값은 "0" 이고 암호화 하지 않음, "1" 로 설정 시 암호화합니다.
file_extension_detector  기본값은 "0" 이고 이미지, 파일등 업로드 시 확장자 변조 체크 하지 않음,
                              "1" 로 설정 시 업로드 시 확장자 변조를 체크합니다.

sample code

에디터에서 파일을 업로드 할 경우 파라미터 암호화 및 확장자 변조를 체크하도록 설정합니다.

XML

.....
<setting>
    .....
    <security>
        <encrypt_param>1</encrypt_param>
        <file_extension_detector>1</file_extension_detector>
    </security>
    .....
</setting>
.....
