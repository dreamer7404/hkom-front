[IE 브라우저 전용]

DEXT5 Editor :: IeBugFixedHyfont

XML : <ie_remove_hyfont>.....</ie_remove_hyfont>
Config : config.IeBugFixedHyfont
XML : <ie_remove_hyfont replace_font="....."></ie_remove_hyfont>
Config : config.IeBugFixedHyfontReplaceFont

Config 설정이 우선 적용됩니다.
IE11브라우저에서 에디터 사용 시 HY폰트의 깨짐 현상을 보정합니다.
hy폰트체의 &nbsp;가 깨져보이는 현상 때문에 폰트를 삭제하거나 대체합니다.

remarks

ie_remove_hyfont  기본값은 "0" 이고 보정 안함, "1"로 설정 시 보정합니다.
replace_font  기본값은 "" 이고 삭제, 글꼴명 설정 시 해당 글꼴로 교체합니다.

sample code

IE11브라우저에서 에디터 사용 시 HY폰트의 깨짐 현상을 보정합니다.

XML

.....
<setting>
    .....
    <browser_bugfixed>
        <ie_remove_hyfont replace_font="굴림">1</ie_remove_hyfont>
    </browser_bugfixed>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.IeBugFixedHyfontReplaceFont = "굴림";

    </script>
