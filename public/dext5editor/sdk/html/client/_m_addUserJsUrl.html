DEXT5 Editor :: addUserJsUrl

void addUserJsUrl(url, editor_id)

에디터 편집영역 iframe document의 head 영역에 해당 url의 js 파일을 import 합니다.

return value

없음.

parameters

url  추가할 js파일의 경로를 의미합니다.
editor_id  js파일을 추가할 에디터의 id를 의미합니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 xss_protection 설정의 xss_use 속성값이 1인 경우,
xss_allow_url 설정에 allow_url로 해당 url을 추가해야 적용됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        // id가 editor1인 에디터의 편집영역 iframe document의 head 영역에 해당 url의 js 파일을 추가합니다.
        var url = 'http://www.dext5.com/dext5editor/js/editor.js'
        DEXT5.addUserJsUrl(url, 'editor1');
    }
        
</script>

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
  
    <!-- 에디터 생성 -->
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>       

</div>

<!-- ..... 생략 ..... -->

</body>
