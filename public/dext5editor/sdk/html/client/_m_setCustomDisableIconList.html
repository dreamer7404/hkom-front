DEXT5 Editor :: setCustomDisableIconList

void setCustomDisableIconList(disableIconList, editor_id)

에디터의 아이콘을 활성화/비활성화 합니다.

return value

없음.

parameters

disableIconList  비활성화할 Icon명의 Sring을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

disableIconList에 기능 및 아이콘명을 콤마로 구분하여 입력시 해당 기능 및 아이콘을 비활성화 합니다.
disableIconList가 공백일시 모든 아이콘 및 기능을 활성화 합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Set(){
        // 비활성화 할 아이콘명 설정
        var disableIcon = "emoticon,image_create,table_insert";
        // id가 editor1인 에디터의 이모티콘, 이미지생성, 표삽입 아이콘을 비활성화 합니다.
        DEXT5.setCustomDisableIconList(disableIcon, 'editor1');
        // work something ...
    }

</script>	

<!-- ..... 생략 ..... -->

    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
