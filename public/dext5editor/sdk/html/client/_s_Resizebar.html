DEXT5 Editor :: Resizebar

XML : <resizebar>.....</resizebar>, Config : config.ResizeBar

에디터 높이를 조절 기능 사용여부를 설정합니다.

remarks

기본값은 "0" 이고 높이 조절 미사용, "1"로 설정 시 에디터 높이 조절 사용으로 설정합니다.

sample code

에디터 높이를 조절 기능 사용여부를 사용함으로 설정합니다.

XML

.....
<setting>
    .....
    <resizebar>1</resizebar>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ResizeBar = "1";

        new Dext5editor("editor1");

    </script>       
</div>
