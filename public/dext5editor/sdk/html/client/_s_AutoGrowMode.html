DEXT5 Editor :: AutoGrowMode

XML : <auto_grow_mode>.....</auto_grow_mode>, Config : config.AutoGrowMode

Config 설정이 우선 적용됩니다.
붙여넣기시 에디터 영역의 크기 초과시 자동으로 확장되는 AutoGrowMode 활성화 여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 AutoGrowMode를 사용합니다.

sample code

AutoGrowMode를 사용합니다.

XML

.....
<plugin_setting>
    <auto_grow_mode>1</auto_grow_mode>
    .....
</plugin_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AutoGrowMode = "1";

        new Dext5editor("editor1");

    </script>       
</div>
