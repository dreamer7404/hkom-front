DEXT5 Editor :: setElementInnerHTML

void setElementInnerHTML(id, strHtml, editor_id)

에디터 body영역 안의 해당 id를 가진 element의 innerHtml을 파라미터의 strHtml로 변경합니다.

return value

에러 발생시에만 에러 문구를 반환합니다.

parameters

id  Element의 id를 의미합니다.
strHtml  변경할 str문구를 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

없음.
