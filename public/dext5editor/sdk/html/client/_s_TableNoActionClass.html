DEXT5 Editor :: TableNoActionClass

XML : <no_action_class>.....</no_action_class>, Config : config.TableNoActionClass

Config 설정이 우선 적용됩니다.
설정하신 class를 갖고 있는 table에 병합/나누기/삭제 등의 테이블 관련 동작을 하지 않도록 설정합니다.

remarks

테이블 동작을 시키지 않도록 설정 할 table객체의 class명을 입력합니다.

sample code

특정 class의 tabel 객체를 동작하지 않도록 설정합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <no_action_class>.....</no_action_class>
        .....
    </table_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
