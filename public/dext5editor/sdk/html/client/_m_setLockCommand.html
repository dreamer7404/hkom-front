DEXT5 Editor :: setLockCommand

void setLockCommand(cmd, lock, editor_id)

특정 액션(cmd)이 실행되지 않도록 합니다.

return value

없음.

parameters

cmd  실행되지 않을 기능을 의미합니다. (기능명이 콤마로 구분된 String 입니다.)
lock  실행 잠금의 활성화/비활성화 여부를 의미합니다. (true||false)
editor_id  에디터의 id를 의미합니다.

remarks

없음.
