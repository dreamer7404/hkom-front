DEXT5 Editor :: ShowFontReal

XML : <show_font_real>.....</show_font_real>Config : config.ShowFontReal
XML : <show_font_real mode="....."></show_font_real>Config : config.ShowFontRealMode

Config 설정이 우선 적용됩니다.
에디터가 사용되는 곳에서의 실제 폰트명 및 크기를 설정합니다.

remarks

show_font_real   기본값은 "1" 이며 설정 시 실제 폰트명, 크기 사용함, "0" 으로 설정 시 사용하지 않습니다.
mode   기본값은 "1" 이며 기존로직 사용, "2"로 설정 시 셀렉션 후 셀렉션 된 부분을 클릭 시 폰트종류, 크기 부분이 잘못 나오는 경우를 보정합니다.

sample code

에디터가 사용되는 곳에서의 실제 폰트명 및 크기를 사용으로 설정합니다.

XML

.....
<setting>
    .....
    <show_font_real mode="2">1</show_font_real>
    .....
</setting>
.....
