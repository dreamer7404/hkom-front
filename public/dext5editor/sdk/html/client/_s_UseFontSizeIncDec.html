DEXT5 Editor :: UseFontSizeIncDec

XML : <font_size use_inc_dec="....."></font_size>, Config : config.UseFontSizeIncDec

Config 설정이 우선 적용됩니다.
tool bar의 글자 크기 콤보박스옆 위아래 클릭도구를 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 tool bar의 글자 크기 콤보박스옆 위아래 클릭도구를 사용합니다.

sample code

tool bar의 글자 크기 콤보박스옆 위아래 클릭도구를 사용하도록 설정합니다.

XML

.....
<font_size use_inc_dec="1">1</font_size>
.....
