DEXT5 Editor :: Placeholder

XML : <placeholder content="....."></placeholder>, Config : config.Placeholder.Content
XML : <placeholder font_color="....."></placeholder>, Config : config.Placeholder.FontColor
XML : <placeholder font_family="....."></placeholder>, Config : config.Placeholder.FontFamily
XML : <placeholder font_size="....."></placeholder>, Config : config.Placeholder.FontSize

Config 설정이 우선 적용됩니다.
에디터 Body 내부에 아무글도 작성 안했을 때 표시되는 기본문구의 내용 및 CSS를 설정합니다.

remarks

content  출력할 문구를 설정합니다.
font_color  Placeholder의 폰트 색상을 설정합니다.
font_family  Placeholder의 글꼴을 설정합니다.
font_size  Placeholder의 폰트 크기를 설정합니다.

sample code

에디터 Body 내부에 아무글도 작성 안했을 때 표시되는 기본문구의 내용 및 CSS를 설정합니다.

XML

.....
<setting>
    .....
    <placeholder content="내용을 입력하세요." font_color="rgb(50,100,140)" font_family="Arial" font_size="20pt"></placeholder>  
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.Placeholder = {
            Content : "내용을 입력하세요.",
            FontColor : "rgb(50,100,140)",
            FontFamily : "Arial",
            FontSize : "20pt"
        }

        new Dext5editor("editor1");

    </script>       
</div>
