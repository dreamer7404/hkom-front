DEXT5 Editor :: CustomEvent

XML : <custom_event><hyperlink><ondbclick>.....</ondbclick></hyperlink></custom_event>
Config : config.CustomEventHyperlinkOndbclick
XML : <custom_event><image><ondbclick>.....</ondbclick></hyperlink></custom_event>
Config : config.CustomEventImageOndbclick

Config 설정이 우선 적용됩니다.
각 태그의 class명 또는 alt가 설정된 값과 같으면 더블 클릭시 설정된 값을 command로 하여 dext_editor_custom_action을 호출하도록 설정합니다.

remarks

class명 또는 alt명을 입력합니다.
custom_event.hyperlink.ondbclick  a 태그의 class명 또는 alt가 설정된 값과 같으면
                                         더블클릭시 설정된 값을 command로 하여 dext_editor_custom_action을 호출합니다.
custom_event.image.ondbclick  img 태그의 class명 또는 alt가 설정된 값과 같으면
                                     더블클릭시 설정된 값을 command로 하여 dext_editor_custom_action을 호출합니다.

sample code

각 태그의 class명 또는 alt가 설정된 값과 같으면 더블클릭시 설정된 값을 command로 하여 dext_editor_custom_action을 호출하도록 설정합니다.

XML

.....
<setting>
    .....
    <custom_event>
        <hyperlink>
            <ondbclick>.....</ondbclick>
        </hyperlink>
        <image>
            <ondbclick>.....</ondbclick>
        </image>
    </custom_event>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->

        DEXT5.config.CustomEventHyperlinkOndbclick = ".....";
        DEXT5.config.CustomEventImageOndbclick = ".....";

    </script>       
</div>
