[IE브라우저 전용]

DEXT5 Editor :: RemoveDummyFontTagInPaste

XML : <remove_dummy_font_tag_in_paste>.....</remove_dummy_font_tag_in_paste>
Config : config.RemoveDummyFontTagInPaste

Config 설정이 우선 적용됩니다.
html 내용 복사/붙여넣기 시 IE 브라우저에서 자동으로 생기는 font 태그를 삭제하도록 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1"로 설정시 html 내용 복사/붙여넣기 시 IE 브라우저에서 자동으로 생기는 font 태그를 삭제합니다.

sample code

html 내용 복사/붙여넣기 시 IE 브라우저에서 자동으로 생기는 font 태그를 삭제하도록 설정합니다.

XML

.....
<setting>
    <remove_dummy_font_tag_in_paste>1</remove_dummy_font_tag_in_paste>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.RemoveDummyFontTagInPaste = "1";

        new Dext5editor("editor1");

    </script>       
</div>
