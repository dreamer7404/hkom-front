DEXT5 Editor :: MiniPhotoEditor

XML : <width="....."></width>
Config : config.MiniPhotoEditor.Width
XML : <height="....."></height>
Config : config.MiniPhotoEditor.Height
XML : <img_tag_remove_attribute="....."></img_tag_remove_attribute>
Config : config.MiniPhotoEditor.ImgTagRemoveAttribute
XML : <background_color="....."></background_color>
Config : config.MiniPhotoEditor.BackgroundColor
XML : <background_opacity="....."></background_opacity>
Config : config.MiniPhotoEditor.BackgroundOpacity

Config 설정이 우선 적용됩니다.
미니 이미지 편집기의 각종 설정을 적용합니다.

remarks

mini_photo_editor.width  미니 이미지 편집기 팝업의 가로 길이를 설정합니다.
mini_photo_editor.height  미니 이미지 편집기 팝업의 세로 길이를 설정합니다.
mini_photo_editor.img_tag_remove_attribute  미니 이미지 편집기로 편집 시 특정 프로퍼티를 removeAttribute 하도록 설정합니다.                                                                                        (콤마(,)로 구분하여 attribute명 입력)
mini_photo_editor.background_color  미니 이미지 편집기 팝업의 뒷 배경 색상을 설정합니다.
mini_photo_editor.background_opacity  미니 이미지 편집기 팝업의 뒷 배경 투명도를 설정합니다. (0.0~1.0)

sample code

미니 이미지 편집기의 각종 설정을 적용합니다.

XML

.....
<setting>
    .....
    <mini_photo_editor>
        <width>800px</width>
        <height>600px</height>
        <img_tag_remove_attribute>alt, class</img_tag_remove_attribute>
        <background_color>rgb(50,100,140)</background_color>
        <background_opacity>0.7</background_opacity>
    </mini_photo_editor>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.MiniPhotoEditor = {
            Width : "800px",
            Height : "600px",
            ImgTagRemoveAttribute : "alt, class",
            BackgroundColor : "rgb(50,100,140)",
            BackgroundOpacity : "0.7"
        }

        new Dext5editor("editor1");

    </script>       
</div>
