DEXT5 Editor :: PasteWhenTableIsLast

XML : <paste_when_table_is_last>.....</paste_when_table_is_last>, Config : config.PasteWhenTableIsLast

Config 설정이 우선 적용됩니다.
붙여넣기 시 테이블이 마지막에 있으면 아래에 p태그를 추가하여 추가된 p태그에 커서가 가도록 합니다.

remarks

기본값은 "1" 이며 붙여넣기 시 테이블이 마지막에 있으면 아래에 p태그를 추가하여 추가된 p태그에 커서가 가고,
"0" 으로 설정 시 해당 기능을 사용하지 않습니다.

sample code

붙여넣기 시 테이블이 마지막에 있으면 아래에 p태그를 추가하여 추가된 p태그에 커서가 가도록 설정합니다.

XML

.....
<setting>
    <paste_when_table_is_last>1</paste_when_table_is_last>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PasteWhenTableIsLast = "1";

        new Dext5editor("editor1");

    </script>       
</div>
