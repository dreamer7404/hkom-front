DEXT5 Editor :: InsertMultiImage

XML : <insert_multi_image>.....</insert_multi_image>, Config : config.InsertMultiImage

Config 설정이 우선 적용됩니다.
멀티 이미지 삽입 기능 사용 여부를 설정합니다.
이미지 삽입 시 여러 장의 이미지를 한번에 선택하여 삽입이 가능하도록 합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 이미지 삽입 시 여러 장의 이미지를 한번에 선택하여 삽입 가능하도록 합니다.

sample code

이미지 삽입 시 여러 장의 이미지를 한번에 선택하여 삽입 가능하도록 합니다.

XML

.....
<setting>
    <insert_multi_image>1</insert_multi_image>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.InsertMultiImage = "1";

        new Dext5editor("editor1");

    </script>       
</div>
