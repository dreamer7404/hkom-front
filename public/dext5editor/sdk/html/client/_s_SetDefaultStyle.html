DEXT5 Editor :: SetDefaultStyle

XML : <set_default_style>.....</set_default_style>
Config : config.SetDefaultStyle.Value
XML : <set_default_style dext_set_style="....."></set_default_style>
Config : config.SetDefaultStyle.DextSetStyle
XML : <set_default_style body_id=".....""></set_default_style>
Config : config.SetDefaultStyle.BodyId
XML : <set_default_style line_height_mode="....."></set_default_style>
Config : config.SetDefaultStyle.LineHeightMode
XML : <set_default_style remove_body_style_in_set="....."></set_default_style>
Config : config.SetDefaultStyle.RemoveBodyStyleInSet
XML : <set_default_style user_style="....."></set_default_style>
Config : config.SetDefaultStyle.UserStyle

Config 설정이 우선 적용됩니다.
set_default_style  에디터 로드 시 기본 스타일이 없을 경우 body 영역에 에디터 스타일을 설정합니다.
dext_set_style  기본 스타일 사용으로 설정 시 DEXT5Editor 기본 CSS 로드 사용 여부를 설정합니다.
body_id  에디터 body 영역에 id 값을 설정합니다.
line_height_mode  get API 사용시 줄간격 값을 상속받은 값으로 설정합니다.
remove_body_style_in_set  set API 사용시 body 태그에 다른 style이나 class가 있으면 제거합니다.
user_style  사용자의 CSS를 사용합니다.

remarks

set_default_style  기본값은 "0" 이고 에디터 기본 스타일 사용,
"1"로 설정 시 에디터 body영역만 스타일 설정,
"2"로 설정 시 getAPI 사용으로 태그에 inline 스타일 설정됩니다.
dext_set_style  기본값은 "1" 이며 스타일 적용, "0" 이면 스타일을 미적용 합니다.
set_default_style 설정값이 "0" 이고,
dext_set_style 설정값을 "0" 으로 설정하면 외부 URL 로드 시 DEXT5Editor 기본 CSS를 로드하지 않습니다.
body_id  기본값은 "dext_body"이고, "사용자 설정값"으로 설정합니다.
line_height_mode  기본값은 "1"이며 설정 사용, "0" 이면 설정을 미적용합니다.
remove_body_style_in_set  기본값은 "0"이며 설정 사용안함, "1" 이면 설정을 사용합니다.
user_style  CSS URL 주소로 설정합니다.

sample code

에디터 로드 시 기본 스타일이 없을 경우 body 영역에 에디터 스타일을 getAPI 사용으로 태그에 inline 스타일 설정합니다.
기본 스타일 사용으로 설정 시 DEXT5Editor 기본 CSS 로드 사용 여부를 사용 안함으로 설정합니다.
에디터 body 영역에 id 값을 "custom_body_id"로 설정합니다.
get API 사용시 줄간격 값을 상속받은 값으로 설정합니다.
set API 사용시 body 태그에 다른 style이나 class가 있으면 제거하도록 설정합니다.

    .....
</setting>
