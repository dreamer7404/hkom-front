Config 설정이 우선 적용됩니다.
ie11브라우저에서 에디터 사용 시 자소분리 현상 및 align 속성 보정 여부를 설정합니다.

remarks

ie11_jaso  기본값은 "0" 이고 보정 안함, "1"로 설정 시 보정합니다.
delete_table_align  기본값은 "0" 이고 보정 안함, "1"로 설정 시 ie11브라우저에서 에디터 사용중 표 align 속성을 제거합니다.
replace_align_margin  기본값은 "0" 이고 보정 안함, "1"로 설정 시 Set API 사용에서 설정에 의해 align 속성이 제거되면 margin 값으로 대체합니다.
replace_br  기본값은 "0" 이고 보정 안함, "1"로 설정 시 get 할 때에 빈 span에 &nbsp;를 삽입합니다.

sample code

ie11브라우저에서 에디터 사용 시 자소분리 현상 및 align 속성 보정 여부를 설정합니다.

XML

.....
<setting>
    .....
    <browser_bugfixed>
        <browser_bugfixed delete_table_align="1" replace_align_margin="1" replace_br="1">1</browser_bugfixed>
    </browser_bugfixed>
    .....
</setting>
.....
