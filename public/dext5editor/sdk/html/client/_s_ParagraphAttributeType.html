DEXT5 Editor :: ParagraphAttributeType

XML : <paragraph_attribute_type>.....</paragraph_attribute_type>, Config : config.ParagraphAttributeType

Config 설정이 우선 적용됩니다.
문단속성 항목의 들여쓰기, 여백의 길이 단위를 설정합니다.

remarks

기본값은 "px" 이며, 길이 단위를 콤마(,)로 구분하여 입력합니다.

sample code

문단속성 항목의 들여쓰기, 여백의 길이 단위 선택항목를 px, mm, cm로 설정합니다.

XML

.....
<setting>
    <paragraph_attribute_type>px,mm,cm</paragraph_attribute_type>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ParagraphAttributeType = "px,mm,cm";

        new Dext5editor("editor1");

    </script>       
</div>
