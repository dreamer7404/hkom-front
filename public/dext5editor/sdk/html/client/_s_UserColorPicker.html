DEXT5 Editor :: UserColorPicker

XML : <user_color_picker>.....</user_color_picker>, Config : config.UserColorPicker

Config 설정이 우선 적용됩니다.
사용자 정의 color picker를 설정합니다.

remarks

색상(HEX, color name)을 콤마(,)로 구분하여 입력합니다.
글꼴색, 글자배경색 선택창에 적용됩니다.

sample code

사용자 정의 color picker를 설정합니다.

XML

.....
<setting>
    <user_color_picker>#32648c,#64648c,#96648c</user_color_picker>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UserColorPicker = "#32648c,#64648c,#96648c";

        new Dext5editor("editor1");

    </script>       
</div>
