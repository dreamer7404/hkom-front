DEXT5 Editor :: findWord

void findWord(option, editor_id)

option 객체의 정보로 에디터 body영역에서 문자열을 찾아 해당 영역을 select 합니다.

return value

없음.

parameters

option  API 설정을 포함한 Object 입니다.
  ({ searchText : "찾을문자열", caseSensitive : 대소문자구분여부(T||F), revertEndDocument : 문서끝에서되돌리기(T||F),
   wordByWord : "단어단위탐색"(T||F), searchDirection : 찾는방향(T:위->아래, F:아래->위), initFocus : 커서 위치 고정여부(T||F) })
editor_id  에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function findWordFunction(){
        var Obj = {
            searchText:'DEXT5', 
            caseSensitive:false, 
            revertEndDocument:true,
            wordByWord:false, 
            searchDirection:true, 
            initFocus:false
        }
        // "DEXT5"라는 문자열을 id가 editor1인 에디터에서 위에서 아래방향으로 문서 끝이면 되돌리고
        // 찾은 위치로 커서를 변경하면서 대소문자 구분없이 문자단위로 찾습니다.
        DEXT5.findWord(Obj,'editor1');
    }

</script>
