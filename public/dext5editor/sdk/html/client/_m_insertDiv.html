DEXT5 Editor :: insertDiv

void insertDiv(attributesObj, divObj, editor_id)

에디터 선택영역에 attributesObj의 정보로 div를 생성하여 삽입합니다.
divObj의 오브젝트가 에디터의 body에 존재하면 해당 div를 attributesObj의 정보로 변경합니다.

return value

없음.

parameters

attributesObj  생성 또는 교체할 div의 정보를 의미합니다.
divObj  에디터 body에 존재하는 교체대상의 div를 의미합니다. (생성을 원하면 null 입력)
editor_id  에디터의 id를 의미합니다.

remarks

attributesObj는 다음과 같은 정보를 설정합니다.
name, id, class, title, alt, style.width, style.height, style.text_align

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Input(){
        // 테스트용 div 생성정보
        var Obj = {
            name:"DEXT5", 
            id:"DEXT5EDITOR", 
            class:"DEXT5CLASS", 
            title:"DEXT5TITLE",
            alt:"dext5",
            "style.width":"200px", 
            "style.height":"100px", 
            "style.text_align":"center"
        };
        // id가 'editor1'인 Obj정보의 div를 새로 생성하여 삽입합니다.
        DEXT5.insertDiv(Obj, null, 'editor1');
        // work something ...
    }

</script>
