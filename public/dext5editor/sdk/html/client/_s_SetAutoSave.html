Config 설정이 우선 적용됩니다.
에디터에서 자동저장에 관련된 기능을 설정합니다.

remarks

set_auto_save  기본값은 "2" 이고 에디터 환경설정에서 사용자가 설정, "0"으로 설정 시 사용 안함, "1"로 설정 시 자동저장 기능을 사용합니다.
interval  자동저장 주기를 숫자만 입력하여 설정합니다. (단위는 "분" 입니다.)
max_count  자동저장될 최대 개수를 숫자만 입력하여 설정합니다.
save_and_start_interval  에디터 환경설정에서 자동저장 기능 활성화 시 최초 저장 여부를 설정합니다.(0:즉시 최초저장, 1:interval후 최초 저장)
unique_key  자동저장되는 localstorage key에 해당 값을 추가합니다. (dext_id_unique_key_0 형태로 저장됩니다.)
use_encrypt  localstorage에 자동 저장되는 html의 암호화 여부를 설정합니다. (0:암호화안함, 1:암호화함)
use_manually_save  수동 임시 저장 기능의 사용여부를 설정합니다. (0:사용안함, 1:사용함)
use_manually_save_shortcut_key  에디터에서 ctrl + s 단축키 입력시 수동 임시 저장을 수행하도록 설정합니다.
                                         (단축키의 기본값은 에디터body를 html로 다운로드하는 기능을 수행합니다.)

sample code

에디터에서 자동저장에 관련된 설정을 적용합니다.

XML

.....
<setting>
    .....
    <set_auto_save interval="10" max_count="3" save_and_start_interval="0" unique_key="autosaved"
    use_encrypt="1" use_manually_save="1" use_manually_save_shortcut_key ="1">1</set_auto_save>
    .....
</setting>
.....
