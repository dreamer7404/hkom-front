DEXT5 Editor :: TableNearCellBorderStyle

XML : <near_cell_border_style>.....</near_cell_border_style>, Config : config.TableNearCellBorderStyle

Config 설정이 우선 적용됩니다.
테이블 근접 셀에 border 스타일을 적용하도록 설정합니다.

remarks

기본값은 "0"으로 설정하지 않음, "1"로 설정시 테이블 근접 셀에 border 스타일을 적용합니다.
(IE브라우저에서 테두리가 보이지 않는 현상이 발생할 경우 근접 셀에 border 스타일을 적용합니다.)

sample code

테이블 근접 셀에 border 스타일을 적용합니다.

XML

.....
<setting>
    .....
    <table_property>
        .....
        <near_cell_border_style>1</near_cell_border_style>
        .....
    </table_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.TableNearCellBorderStyle = "1";

        new Dext5editor("editor1");

    </script>       
</div>
