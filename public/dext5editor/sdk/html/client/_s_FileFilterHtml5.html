DEXT5 Editor :: FileFilterHtml5

XML : <file_filter_html5>.....</file_filter_html5>, Config : config.FileFilterHtml5

Config 설정이 우선 적용됩니다.
웹표준 모드에서 이미지 삽입 dialog 창의 확장자 선택을 설정합니다.

remarks

웹표준 모드에서 이미지 삽입 dialog 창의 확장자를 .확장자1,.확장자2.. 으로 설정합니다.

sample code

웹표준 모드에서 이미지 삽입 dialog 창의 확장자 선택을 설정합니다. 

XML

.....
<setting>
    <file_filter_html5>.jpg,.png</file_filter_html5>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.FileFilterHtml5 = ".jpg,.png";

        new Dext5editor("editor1");

    </script>       
</div>
