DEXT5 Editor :: dext5EventBeforeOpenHtmlConvert

void dext5EventBeforeOpenHtmlConvert(strFileFormat, strFileFullPath)

플러그인 모드 에디터에서 파일열기 수행 전 이벤트가 발생합니다.

return value

없음.

parameters

strFileFormat  열기한 파일의 파일포맷을 의미합니다.
strFileFullPath  열기한 파일의 실제 경로를 의미합니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 setting 설정 open_document의 before_open_event 속성값이 1인 경우 이벤트가 발생합니다.
