DEXT5 Editor :: UseRuler

XML : <use_ruler mode="2"></use_ruler>, Config : config.Ruler.Mode
XML : <use_ruler move_guide_line_color="....."></use_ruler>, Config : config.Ruler.MoveGuideLineColor
XML : <use_ruler move_guide_line_style="....."></use_ruler>, Config : config.Ruler.MoveGuideLineStyle
XML : <use_ruler use_inoutdent="....."></use_ruler>, Config : config.Ruler.UseInoutdent
XML : <use_ruler move_gap="....."></use_ruler>, Config : config.Ruler.MoveGap
XML : <use_ruler use_resize_event="....."></use_ruler>, Config : config.Ruler.UseResizeEvent
XML : <use_ruler use_mouse_guide="....."></use_ruler>, Config : config.Ruler.UseMouseGuide

Config 설정이 우선 적용됩니다.
XML:mode,Config:Mode  눈금자의 모드를 설정합니다.
아래 설정들은 use_ruler의 mode가 "2"일 때 적용됩니다.
XML:move_guide_line_color,Config:MoveGuideLineColor  움직이는 가이드라인의 색상을 설정합니다.
XML:move_guide_line_style,Config:MoveGuideLineStyle  움직이는 가이드라인의 스타일을 설정합니다.
XML:use_inoutdent,Config:UseInoutdent  가이드라인을 움직였을때 들여쓰기 및 내어쓰기 사용 여부를 설정합니다.
XML:move_gap,Config:MoveGap  가이드라인을 움직일때 눈금자의 이동 간격을 설정합니다.
XML:use_resize_event,Config:UseResizeEvent  에디터 사이즈 변경시 눈금자의 위치를 다시 잡아주는 기능을 설정합니다.
XML:use_mouse_guide,Config:UseMouseGuide  에디터 body영역에서 마우스를 움직일시 세로 눈금자에 위치를 표시하는 마우스 가이드 사용 여부를 설정합니다.

remarks

XML:mode,Config:Mode  기본값은 "1" 이고 일반자 모드로 설정, "2"로 설정 시 가이드라인 위치 조정이 가능한 모드로 설정합니다.
아래 설정들은 use_ruler의 mode가 "2"일 때 적용됩니다.
XML:move_guide_line_color,Config:MoveGuideLineColor  기본값은 #666666 입니다. (색상의 HEX, rgb 등으로 설정 가능합니다.)
XML:move_guide_line_style,Config:MoveGuideLineStyle  기본값은 dotted 입니다. (border-style css값으로 설정 가능합니다.)
XML:use_inoutdent,Config:UseInoutdent  기본값은 "1" 이고 가이드라인 조정시 들여쓰기 및 내어쓰기 사용함, "0" 으로 설정 시 사용하지 않습니다.
XML:move_gap,Config:MoveGap  기본값은 "1" 이고 가이드라인 조정시 눈금자의 이동 간격을 설정합니다.
XML:use_resize_event,Config:UseResizeEvent  기본값은 "1" 이고 에디터 사이즈 변경시 눈금자의 위치를 다시 잡도록 설정,
"0" 으로 조정 안함으로 설정합니다.
XML:use_mouse_guide,Config:UseMouseGuide  기본값은 "0" 이고 사용 안함, "
1" 로 설정 시 에디터 body영역에서 마우스를 움직일시 세로 눈금자에 위치를 표시하도록 설정합니다.

sample code

눈금자 모드를 가이드라인 위치 조정이 가능한 모드("2")로 설정합니다. 
가이드라인을 옮길때 움직이는 가이드라인의 색상을 #328cc8로 설정합니다.
가이드라인을 옮길때 움직이는 가이드라인의 border-style을 solid로 설정합니다.
가이드라인의 위치 조정시 들여쓰기 및 내어쓰기를 사용하도록 설정합니다.
가이드라인의 위치 조정시 눈금자 이동 간격을 10으로 설정합니다.
에디터 사이즈 변경시 눈금자의 위치를 다시 잡도록 설정합니다.
에디터 body영역에서 마우스를 움직일시 세로 눈금자에 위치를 표시하도록 설정합니다. 

XML

.....
<setting>
    .....
    <use_ruler mode="2" move_guide_line_color="#328cc8" move_guide_line_style="solid" use_inoutdent="1"
    move_gap="10" use_resize_event="1" use_mouse_guide="1">1</use_ruler>
    .....
</setting>
.....
