DEXT5 Editor :: getPersonalDataValidation

Boolean getPersonalDataValidation(editor_id)

에디터의 작성영역에 개인정보(주민등록번호, 이메일, 전화번호 등)가 존재하는지를 확인합니다.

return value

해당 에디터 작성영역에 개인정보 존재 여부

parameters

editor_id  에디터의 id를 의미합니다.

remarks

주민등록번호, 이메일, 전화번호, custom 정보로 설정된 항목만 체크합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>

        var dext5 = DEXT5.getPersonalDataValidation('editor1');
        alert(dext5);
        // work something ...
    }

</script>
