DEXT5 Editor :: replaceBlocktoBr

String replaceBlocktoBr(htmlString)

html 소스의 p 태그를 삭제 후 br 태그로 변경한 후 반환합니다.

return value

p 태그가 br로 치환된 html 소스.

parameters

htmlString  변경할 html 소스를 의미합니다.

remarks

없음.
