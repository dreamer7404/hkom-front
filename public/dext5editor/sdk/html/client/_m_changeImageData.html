DEXT5 Editor :: changeImageData

void changeImageData(valueObject, editor_id)

에디터 body영역에서 선택되어있는 image element를 파라미터의 image element로 변경합니다.

return value

없음.

parameters

valueObject  에디터 body에서 선택된 이미지를 대체할 이미지 element를 의미합니다.
editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

valueObject는 다음과 같은 정보를 설정합니다.
src, style.width, style.height

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_mouse_event(mouseEvent, eventObject, currElem) {
        // img element를 mouseup 시에 적용되도록 설정합니다.
        if (mouseEvent == "mouseup" && currElem.nodeName == "IMG") {
            // 테스트용 이미지 element를 생성합니다.
            var tempImg = document.createElement('IMG');
            tempImg.src = "http://www.raonwiz.com/images/raon_logo.png";
            tempImg.width = 200;
            tempImg.height = 100;
            // mouseup 한 element 객체와 테스트용 이미지 element의 src가 같지 않으면 적용합니다.
            if (currElem.src != tempImg.src) {
                // editor1의 id를 가진 에디터에 img element가 선택상태이면 tempImg로 교체합니다.
                DEXT5.changeImageData(tempImg, 'editor1');
            }
        }
    }
        
</script>
