DEXT5 Editor :: FileFilterPlugin

XML : <file_filter_plugin>.....</file_filter_plugin>, Config : config.FileFilterPlugin

Config 설정이 우선 적용됩니다.
플러그인 모드에서 이미지 삽입 dialog 창의 확장자 선택을 설정합니다.

remarks

플러그인 모드에서 이미지 삽입 dialog 창의 확장자를 확장자 이미지|세부 확장자1,세부 확장자2.. 으로 설정합니다.

sample code

플러그인 모드에서 이미지 삽입 dialog 창의 확장자를 설정합니다.

XML

.....
<setting>
    <file_filter_plugin>bpm 이미지|bmp|JPG,PNG Images|jpg,jpeg,png</file_filter_plugin>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.FileFilterPlugin = "bpm 이미지|bmp|JPG,PNG Images|jpg,jpeg,png";

        new Dext5editor("editor1");

    </script>       
</div>
