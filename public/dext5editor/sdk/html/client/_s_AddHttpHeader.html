DEXT5 Editor :: AddHttpHeader

config.AddHttpHeader

request header를 설정하여 추가합니다.

remarks

리스트에 헤더 값을 입력합니다.
헤더명은 리스트의 인덱스순서의 숫자로 부여됩니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        // 요청 헤더에 0 : DEXT5Editor 를 추가합니다.
        DEXT5.config.AddHttpHeader = ["DEXT5Editor"];

        new Dext5editor("editor1");
    </script>       
</div>
