DEXT5 Editor :: IgnoreDifferentEditorName

XML : <ignore_different_editor_name>.....</ignore_different_editor_name>
Config : config.IgnoreDifferentEditorName

Config 설정이 우선 적용됩니다.
에디터를 하나만 생성한 이후 API 사용중 아이디를 잘못 넣을 경우 API Call 오류가 발생하지 않도록 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1"로 설정 시 에디터를 하나만 생성한 이후 API 사용중 아이디를 잘못 넣을 경우 API Call 오류가 발생하지 않도록 설정합니다.

sample code

            에디터를 하나만 생성한 이후 API 사용중 아이디를 잘못 넣을 경우 API Call 오류가 발생하지 않도록 설정합니다.

.....
<setting>
    <ignore_different_editor_name>1</ignore_different_editor_name>
