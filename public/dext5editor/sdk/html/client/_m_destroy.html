DEXT5 Editor :: destroy

void destroy(editor_id, noRecycling)

해당 id의 에디터를 제거합니다.

return value

없음.

parameters

editor_id  API를 적용할 에디터의 id를 의미합니다.
noRecycling  DEXT5 객체의 재사용 여부를 의미합니다. (true:재사용안함, false:재사용함)

remarks

없음.

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

        // id가 editor1인 에디터를 삭제하고 DEXT5 객체는 제거하지 않습니다.
        DEXT5.destroy('editor1', false);
    }
        
</script>
