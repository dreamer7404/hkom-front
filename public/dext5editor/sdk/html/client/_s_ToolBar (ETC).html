DEXT5 Editor :: ToolBar (ETC)

XML : <tool top=".....">full_screen</tool>, Config : config.FullScreenTop
XML : <tool bottom=".....">full_screen</tool>, Config : config.FullScreenBottom
XML : <tool left=".....">full_screen</tool>, Config : config.FullScreenLeft
XML : <tool right=".....">full_screen</tool>, Config : config.FullScreenRight
XML : <tool start_main_page=".....">help</tool>, Config : config.HelpStartMainPage

Config 설정이 우선 적용됩니다.
전체화면시 에디터 전체영역의 inset값(top, right, bottom, left)을 설정합니다.
help start page를 dext5 소개 페이지로 할지의 여부를 설정합니다.

remarks

full_screen top   전체화면시 에디터 전체영역의 top inset값을 설정합니다.
full_screen bottom   전체화면시 에디터 전체영역의 bottom inset값을 설정합니다.
full_screen left   전체화면시 에디터 전체영역의 left inset값을 설정합니다.
full_screen right   전체화면시 에디터 전체영역의 right inset값을 설정합니다.
help start_main_page   기본값은 "1"로 사용자 도움말 기본 페이지를 dext5 소개 페이지로 설정하고,
                            "0"으로 설정시 우측 탭의 맨 처음 탭을 기본 페이지로 설정합니다.

sample code

전체화면시 에디터 전체영역의 inset값을 200px 200px 200px 200px로 설정합니다.
사용자 도움말 기본 페이지를 dext5 소개 페이지로 설정합니다.

XML

.....
<tool_bar_1>
    .....
    <tool left="200" top="200" right="200" bottom="200">full_screen</tool>
    <tool start_main_page="1">help</tool>
    .....
</tool_bar_1>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        DEXT5.config.ToolBar1.FullScreenTop = "200";
        DEXT5.config.ToolBar1.FullScreenBottom = "200";
        DEXT5.config.ToolBar1.FullScreenLeft = "200";
        DEXT5.config.ToolBar1.FullScreenRight = "200";
        DEXT5.config.HelpStartMainPage = "1";

        new Dext5editor("editor1");

    </script>       
</div>
