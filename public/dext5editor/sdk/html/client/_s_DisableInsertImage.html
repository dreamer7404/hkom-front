DEXT5 Editor :: DisableInsertImage

XML : <disable_insert_image>.....</disable_insert_image>, Config : config.DisableInsertImage

Config 설정이 우선 적용됩니다.
모든 이미지를 삽입(업로드)하는 기능을 제한하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않고,
"1" 로 설정 시 모든 이미지를 삽입(업로드)하는 기능을 제한합니다.
이미지삽입, 문서배경, 표배경, 셀배경에 적용됩니다.

sample code

모든 이미지를 삽입(업로드)하는 기능을 제한합니다.

XML

.....
<setting>
    <disable_insert_image>1</disable_insert_image>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">

        DEXT5.config.DisableInsertImage = "1";

    </script>
