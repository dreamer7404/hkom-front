[IE 브라우저 전용]

DEXT5 Editor :: Ie11BugFixedTypingBugInTable

XML : <ie11_typing_bug_in_table>.....</ie11_typing_bug_in_table>
Config : config.Ie11BugFixedTypingBugInTable

Config 설정이 우선 적용됩니다.
IE11 브라우저 사용중 셀 안의 p태그가 비어있을 경우 br 태그를 추가하여 한글 입력이 오동작 하는 현상을 보정합니다.

remarks

Ie11BugFixedJASO가 1일 때 적용됩니다.
기본값은 "0" 이고 반영 안함,
"1"로 설정 시 현상을 보정합니다.

sample code

IE11 브라우저 사용중 셀 안의 p태그가 비어있을 경우 br 태그를 추가하여 한글 입력이 오동작 하는 현상을 보정합니다.

XML

.....
<setting>
    .....
    <browser_bugfixed>
        <ie11_typing_bug_in_table>1</ie11_typing_bug_in_table>
    </browser_bugfixed>
    .....
</setting>
.....
