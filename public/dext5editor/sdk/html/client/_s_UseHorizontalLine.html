DEXT5 Editor :: UseHorizontalLine

XML : <use_horizontal_line>.....</use_horizontal_line>
XML : <use_horizontal_line url="....."></use_horizontal_line>, Config : config.UseHorizontalLine
XML : <use_horizontal_line height="....."></use_horizontal_line>, Config : config.UseHorizontalLineHeight
XML : <use_horizontal_line style="....."></use_horizontal_line>, Config : config.UseHorizontalLineStyle

Config 설정이 우선 적용됩니다.
use_horizontal_line  기준선 표시 사용여부를 설정합니다.
url  기준선 이미지 경로를 설정합니다.
height  에디터 자체 가로 기준선의 사용여부를 설정합니다. (사용시 url의 이미지는 설정하여도 적용되지 않습니다.)
style  UseHorizontalLineHeight="1"로 설정시 적용되며, 기준선의 style(CSS)를 설정합니다.

remarks

url에 적용된 이미지에 따라 가로 또는 세로선으로 커스텀이 가능합니다.
ex)1000x1 하얀배경 이미지 가로 300px 위치에 1px점 하나를 찍으면 300px마다 세로선으로 표현됩니다.
    1x150 하얀배경 이미지 맨 아래에 1px점 하나를 찍으면 150px마다 가로선으로 표현됩니다.

use_horizontal_line  "0" 으로 설정 시 사용 안함, "1"로 설정 시 가로, 세로 기준선 표시 사용으로 설정합니다.
url  http://를 포함한 전체 이미지 경로를 콤마(,)로 구분하여 입력합니다.
height  기본값은 "0"으로 설정 시 사용 안함, "1"로 설정 시 url에 설정된 이미지가 아닌 에디터 자체 가로 기준선을 표시합니다.
style  height "1"로 설정시 적용되며, 에디터 자체 기준선의 lineStyle CSS를 설정합니다. (기본값은 ""으로 1px dotted red가 적용됩니다.)

sample code

세로 기준선 표시 사용여부를 사용으로 설정합니다.
기준선 이미지 경로를 http://www.dext5.com/custom_horizontal_line.png 으로 설정합니다.

XML

.....
<setting>
    .....
    <use_horizontal_line url="http://www.dext5.com/custom_horizontal_line.png">1</use_horizontal_line>
    <!-- 에디터 자체 가로 기준선을 사용하시려면 다음과 같이 설정합니다. (예시:100px마다 3px dotted rgb(50,100,140)로 표시) -->
    <!--<use_horizontal_line height="100" style="3px dotted rgb(50,100,140)">1</use_horizontal_line>-->
    .....
</setting>
.....
