DEXT5 Editor :: UseHtmlProcessByWorker

XML : <use_htmlprocess_worker>.....</use_htmlprocess_worker>, Config : config.UseHtmlProcessByWorker
XML : <use_htmlprocess_worker use_set_api="....."></use_htmlprocess_worker>,
Config : config.UseHtmlProcessByWorkerSetApi

Config 설정이 우선 적용됩니다.
HTML 처리 및 set API 사용시 Web Worker의 사용 여부를 설정합니다.

remarks

use_htmlprocess_worker  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 HTML 처리시 Web Worker를 사용합니다.
use_set_api  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 set API 사용시 Web Worker를 사용합니다.

sample code

HTML 처리 및 set API 사용시 Web Worker를 사용하도록 설정합니다.

XML

.....
<use_htmlprocess_worker use_set_api="1">1</use_htmlprocess_worker>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UseHtmlProcessByWorker = "1";
        DEXT5.config.UseHtmlProcessByWorkerSetApi = "1";

    </script>       
</div>
