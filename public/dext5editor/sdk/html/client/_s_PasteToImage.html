DEXT5 Editor :: PasteToImage

XML : <paste_to_image>.....</paste_to_image> Config : config.PasteToImage
XML : <paste_to_image excel_image_fix="....."></paste_to_image> Config : config.ExcelImageFix
XML : <paste_to_image popup_mode="....."></paste_to_image> Config : config.PasteToImagePopupMode
XML : <paste_to_image type="....."></paste_to_image> Config : config.PasteToImageType

Config 설정이 우선 적용됩니다.
에디터 이미지로 붙여넣기 사용여부 및 각종 항목을 설정합니다.

remarks

paste_to_image   기본값은 "1" 이고 기본 붙여넣기에 이미지로 붙여넣기를 설정, "0" 으로 설정 시 우클릭 이미지로 붙여넣기 항목을 설정합니다.
                     "1" 로 설정시 이미지로 붙여넣기 다이얼로그를 사용, "0" 으로 설정 시 이미지로 붙여넣기 다이얼로그를 사용하지 않습니다.
                     "2" 로 설정시 우클릭 context 및 붙여넣기창을 모두 활성화 합니다.
paste_to_image@excel_image_fix  엑셀 이미지 붙여넣기 시 테이블 일부 테두리가 회색으로 나오는 경우 보정 처리하도록 설정합니다.
paste_to_image@popup_mode  이미지 붙여넣기시 팝업창을 설정합니다. (0=너비/높이 등 크기만 조절 가능, 1=회전, 포맷 등 다양한 설정 가능)
paste_to_image@type  이미지로 붙여넣기 시 png, jpg로 붙여넣기를 설정합니다. (콤마(,)로 구분하여 입력합니다.)

sample code

에디터 이미지로 붙여넣기 사용여부 및 각종 항목을 설정합니다.

XML

.....
<setting>
    .....
    <paste_to_image excel_image_fix="1" popup_mode="1" type="png,jpg">1</paste_to_image>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">  
    <script type="text/javascript">

        DEXT5.config.PasteToImage = "1";
        DEXT5.config.ExcelImageFix = "1";
        DEXT5.config.PasteToImagePopupMode = "1";
        DEXT5.config.PasteToImageType = "png,jpg";

        new Dext5editor("editor1");

    </script>       
</div>
