DEXT5 Editor :: RemoveStyle

XML : <use_remove_style>.....</use_remove_style>, Config : config.RemoveStyle.Use
XML : <use_remove_style style="....."></use_remove_style>, Config : config.RemoveStyle.Style
XML : <use_remove_style tag="....."></use_remove_style>, Config : config.RemoveStyle.Tag

Config 설정이 우선 적용됩니다.
set API 사용시 태그의 스타일 삭제여부 및 삭제 조건을 설정합니다.

remarks

use_remove_style  기본값은 "0" 이며 사용 안함, "1" 로 설정 시 set API 사용시 태그의 스타일을 삭제 하도록 설정합니다.
style  기본값은 "" 이며, style 항목 입력 시 해당 style 항목을 삭제 합니다.
tag  기본값은 "" 이며 모든 태그에 적용함, 태그명 입력시 해당 태그의 style만 삭제 합니다.

sample code

set API 사용시 태그의 스타일 삭제여부 및 삭제 조건을 설정합니다.
