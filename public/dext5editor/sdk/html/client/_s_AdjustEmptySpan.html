DEXT5 Editor :: AdjustEmptySpan

XML : <adjust_empty_span>.....</adjust_empty_span>, Config : config.AdjustEmptySpan

Config 설정이 우선 적용됩니다.
get API 사용시 빈 span 태그에 &nbsp; 또는 br태그를 넣어주는 옵션을 설정합니다.

remarks

기본값은 "1" 이며 &nbsp;를 삽입하고,
"0" 으로 설정 시 br태그를 삽입합니다.

sample code

get API 사용시 빈 span 태그에 &nbsp;를 넣어주도록 설정 합니다.

XML

.....
<setting>
    <adjust_empty_span>1</adjust_empty_span>
    .....
</setting>
.....
