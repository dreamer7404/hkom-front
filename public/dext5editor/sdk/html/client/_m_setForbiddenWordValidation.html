DEXT5 Editor :: setForbiddenWordValidation

void setForbiddenWordValidation(value, editor_id)

금지어 추출 후 다이얼로그로 출력합니다.

return value

없음.

parameters

editor_id  에디터의 id를 의미합니다.

remarks

에디터 설정파일(dext5editor/config/dext_editor.xml)의 forbidden_word에
지정된 금지어가 있을시 다이얼로그에 출력됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5ForbiddenWord(){
        // id가 editor1인 에디터에서 금지어추출 다이얼로그가 활성화 됩니다.
        DEXT5.setForbiddenWordValidation('editor1');
        // work something ...
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px;"> 
 
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
