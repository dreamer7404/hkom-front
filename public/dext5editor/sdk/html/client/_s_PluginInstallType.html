DEXT5 Editor :: PluginInstallType

XML : <plugin_install_type>.....</plugin_install_type>
Config : config.PluginInstallType
XML : <plugin_install_type install_url="....."></plugin_install_type>
Config : config.PluginInstallUrl
XML : <plugin_install_type use_install_guide="....."></plugin_install_type>
Config : config.UsePluginInstallGuide
XML : <plugin_install_type install_guide_type="....."></plugin_install_type>
Config : config.PluginInstallGuideType
XML : <plugin_install_type zIndex="....."></plugin_install_type>
Config : config.PluginInstallGuideZIndex

Config 설정이 우선 적용됩니다.
plugin 설치관련 내용을 설정합니다.

remarks

plugin_install_type  기본값은 "0"이며 설치시 dext5editor.exe를 설치,
"2"로 설정 시 설치시 dext5editorSL.exe를 설치합니다.
install_url  plugin_install_type의 설정값이 "1"일때 설정 되며, 플러그인 설치 url을 설정합니다.
(미설정 시 기본경로(제품폴더>plugin)의 파일을 설치하게 됩니다.)
use_install_guide  플러그인 설치시 설치안내 팝업창의 팝업여부를 설정합니다.
install_guide_type  use_install_guide의 설정값이 "1"일때 설정되며, "0"으로 설정시 플러그인 설치 완료 후에도 설치안내 팝업을 유지하고,
"1"로 설정시 플러그인 설치 완료 후 설치안내 팝업을 닫습니다.
zIndex  install_guide_type의 설정값이 "0"일때 설정되며, 플러그인 설치 시 설치안내 팝업창의 zIndex를 설정합니다. (기본값은 1000000 입니다.)

sample code

plugin 설치관련 내용을 설정합니다.

XML

.....
<setting>
    <plugin_install_type install_url="....." use_install_guide="1" install_guide_type="0" zIndex="500000">0</plugin_install_type>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PluginInstallType = "0";
        DEXT5.config.PluginInstallUrl = ".....";
        DEXT5.config.UsePluginInstallGuide = "1";
        DEXT5.config.PluginInstallGuideType = "0";
        DEXT5.config.PluginInstallGuideZIndex = "500000";
