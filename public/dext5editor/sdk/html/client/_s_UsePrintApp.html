DEXT5 Editor :: UsePrintApp

XML : <use_print_app>.....</use_print_app>, Config : config.UsePrintApp

Config 설정이 우선 적용됩니다.
오픈 소스 프린트 방식 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 오픈 소스 프린트 방식을 사용합니다.

sample code

오픈 소스 프린트 방식을 사용합니다.

XML

.....
<license>
    <use_print_app>1</use_print_app>
    .....
</license>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UsePrintApp = "1";

        new Dext5editor("editor1");

    </script>       
</div>
