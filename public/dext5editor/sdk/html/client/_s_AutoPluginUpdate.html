[플러그인 전용]

DEXT5 Editor :: AutoPluginUpdate

XML : <auto_plugin_update>.....</auto_plugin_update>, Config : config.AutoPluginUpdate

Config 설정이 우선 적용됩니다.
플러그인의 자동 업데이트 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 으로 설정 시 플러그인을 자동 업데이트 하도록 설정합니다.

sample code

플러그인을 자동 업데이트 하도록 설정합니다.

XML

.....
<setting>
    <auto_plugin_update>1</auto_plugin_update>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AutoPluginUpdate = "1";

        new Dext5editor("editor1");

    </script>       
</div>
