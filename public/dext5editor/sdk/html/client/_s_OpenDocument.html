DEXT5 Editor :: OpenDocument

XML : <open_document before_open_event="....."></open_document>
Config : config.OpenDocument.BeforeOpenEvent
XML : <open_document use_html5_fileopen="....."></open_document>
Config : config.OpenDocument.UseHtml5FileOpen
XML : <open_document use_hwp="....."></open_document>, Config : config.OpenDocument.UseHwp

Config 설정이 우선 적용됩니다.
파일 열기 관련 설정을 적용합니다.

remarks

BeforeOpenEvent  파일 오픈전 이벤트(dext5EventBeforeOpenHtmlConvert) 호출여부를 설정합니다. (0 : 이벤트 사용안함, 1 : 이벤트 사용함)
UseHtml5FileOpen  "0"으로 설정시 파일 열기기능 사용시 html파일 open 불가능, "1"로 설정시 가능하도록 설정합니다.
UseHwp  "0"으로 설정시 파일 열기기능 사용시 hwp파일 open 불가능, "1"로 설정시 가능하도록 설정합니다.

sample code

hwp, html 파일 열기를 허용하고 열기전 이벤트를 발생시키도록 설정합니다.

XML

.....
<setting>
    .....
    <open_document before_open_event="1" use_html5_fileopen="1" use_hwp="1">
        .....
    </open_document>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
