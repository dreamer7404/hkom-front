DEXT5 Editor :: PasteImageBase64Remove

XML : <paste_image_base64 remove="....."></paste_image_base64>
Config : config.PasteImageBase64Remove

Config 설정이 우선 적용됩니다.
이미지를 붙여넣기 할 때 base64이미지를 삭제하도록 처리합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 이미지를 붙여넣기 할 때 base64 이미지를 삭제하도록 설정합니다.

sample code

이미지를 붙여넣기 할 때 base64 이미지를 삭제하도록 설정합니다.

XML

.....
<extra_setting>
    <paste_image_base64 remove="1">1</paste_image_base64>
    .....
</extra_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PasteImageBase64Remove = "1";

        new Dext5editor("editor1");

    </script>       
</div>
