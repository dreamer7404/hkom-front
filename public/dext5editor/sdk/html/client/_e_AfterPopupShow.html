DEXT5 Editor :: dext_editor_afterpopupshow_event

void dext_editor_afterpopupshow_event(editorId, targetObj, popupType)

에디터의 모든 팝업시 이벤트가 발생합니다.

return value

없음.

parameters

editorId  팝업이 발생한 에디터의 Id 입니다.
targetObj  이벤트 대상의 객체 입니다.
popupType  발생한 팝업의 타입명 입니다.

remarks

dialog, topmenu, selectbox 등의 팝업시 이벤트가 발생합니다.
