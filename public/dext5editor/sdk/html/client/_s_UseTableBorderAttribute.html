DEXT5 Editor :: UseTableBorderAttribute

XML : <table_property use_border_attribute = "....."></table_property>,
Config : config.UseTableBorderAttribute

Config 설정이 우선 적용됩니다.
표 생성 시 table 태그의 border 속성의 유무를 설정합니다.

remarks

기본값은 "1"로 표 생성시 table 태그의 border="1" 속성을 추가,
"0"으로 설정시 table 태그의 border 속성을 삭제합니다.

sample code

table 태그의 border 속성을 삭제합니다.

XML

.....
<setting>
    .....
    <table_property use_border_attribute="0">
        .....
    </table_property>
    .....
</setting>
.....
