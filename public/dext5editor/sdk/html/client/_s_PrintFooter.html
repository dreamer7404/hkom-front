DEXT5 Editor :: PrintFooter

XML : <print_footer>.....</print_footer>, Config : config.PrintFooter

Config 설정이 우선 적용됩니다.
인쇄 시 Footer 영역의 내용을 설정합니다.

sample code

인쇄 시 Footer 영역의 내용에 "Footer"를 설정합니다.

XML
.....
<dext5>
    <license>
        .....
        <print_footer>Footer</print_footer>
        .....
    </license>
    .....
</dext5>
