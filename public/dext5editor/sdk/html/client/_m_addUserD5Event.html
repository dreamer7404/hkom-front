DEXT5 Editor :: addUserD5Event

string addUserD5Event(id, eventName, funObj, editor_id)

에디터 작성영역에 해당 id를 가진 element가 있으면 이벤트를 추가 합니다.

return value

API 사용이 올바르지 않거나, 파라미터의 id로 존재하는 element가 없을 시 해당하는 문구를 리턴합니다.
적용이 완료되면 '' 을 리턴합니다.

parameters

id  이벤트를 추가할 에디터 작성영역에 존재하는 element의 ID 값을 의미합니다.
eventName  추가할 이벤트의 종류를 의미합니다.
funObj  적용한 element에서 이벤트 발생시 실행할 함수를 의미합니다.
editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        // 테스트용 element(<p>)를 추가합니다.
        var html = '<p id="D5Event">Event Test</p>';
        DEXT5.setInsertHTML(html, 'editor1');
    }

    function dext_editor_set_insert_complete_event() {
        // 에디터에 element가 추가완료되면 test함수를 생성한 후 API를 호출합니다.
        var D5func = function () { alert("Hello DEXT5 Editor!!"); }


        // id가 editor1인 에디터의 body영역에 D5Event의 id를 가진 element가 존재하면 mousedown 이벤트로 D5func를 추가합니다.
        var message = DEXT5.addUserD5Event('D5Event', 'mousedown', D5func, 'editor1');

        
        // 리턴값으로 API 적용 여부를 확인합니다.
        if(message!=''){alert("API적용 실패");}
        else{alert("API적용 완료");}
    }

</script>
