DEXT5 Editor :: ForceSaveAsServer

XML : <force_save_as_server>.....</force_save_as_server>, Config : config.ForceSaveAsServer

Config 설정이 우선 적용됩니다.
파일로 저장시 서버에서 내려주도록 설정합니다.
(클라이언트에서 바로 처리시 로컬 프로그램 충돌로 인하여 브라우저가 다운되는 현상을 방지합니다.)

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 파일로 저장하면 서버에서 내려주도록 설정합니다.

sample code

파일로 저장시 서버에서 내려주도록 설정합니다.

XML

.....
<setting>
    <force_save_as_server>1</force_save_as_server>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ForceSaveAsServer = "1";

        new Dext5editor("editor1");

    </script>       
</div>
