DEXT5 Editor :: doSaveHTMLEx

string doSaveHTMLEx(option, editor_id)

html 파일로 저장하기를 수행 합니다.

return value

html 파일이 저장된 경로

parameters

option  API 설정을 포함한 Object 입니다.
  {
    strHtml:"저장할 html 소스",
    strSavePath:"저장 경로",
    strCharSet:"인코딩",
    bNotOpenDialog:다운로드 dialog open 여부(true||false)
  }
editor_id  에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function saveHTML(){
        // editor1 작성내용의 html을 변수에 저장합니다.
        var html = DEXT5.getBodyValueEx('editor1');
        // strHtml내용의 html을 strSavePath의 경로에 strCharSet의 문자인코딩 방식으로 다운로드 dialog를 띄우지 않고 저장합니다.
        var Obj = {
            strHtml:html, 
            strSavePath:"D:\\temp\\dext5doc.html", 
            strCharSet:"UTF-8", 
            bNotOpenDialog:true
        }
        var savedPath = DEXT5.doSaveHTMLEx(Obj,'editor1');
        alert(savedPath);
    }

</script>
