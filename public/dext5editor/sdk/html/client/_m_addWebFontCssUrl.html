DEXT5 Editor :: addWebFontCssUrl

void addWebFontCssUrl(url, editor_id)

에디터 편집영역 iframe document의 head 영역에 해당 url의 css 파일을 import 하고, 폰트 콤보박스 영역에도 추가합니다.

return value

없음.

parameters

url  추가할 css파일의 경로를 의미합니다.
editor_id  css파일을 추가할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        // id가 editor1인 에디터의 편집영역 iframe document의 head 영역에 해당 url의 css 파일을 추가하고 폰트 콤보박스 영역에도 추가합니다.
        var url = 'http://www.dext5.com/dext5editor/css/editor.css'
        DEXT5.addUserJsUrl(url, 'editor1');
    }
        
</script>
