DEXT5 Editor :: DefaultFontMargin

XML : <font_margin bottom="....."></font_margin>, Config : config.DefaultFontMarginBottom
XML : <font_margin top="....."></font_margin>, Config : config.DefaultFontMarginTop

Config 설정이 우선 적용됩니다.
텍스트 폰트의 위, 아래 기본 여백(margin-top, margin-bottom)을 설정합니다.

remarks

입력값의 수치만큼 텍스트폰트의 위, 아래 기본 여백을 설정합니다.

sample code

텍스트 폰트의 위, 아래 기본 여백을 4px로 설정합니다.

XML

.....
<setting>
    <font_margin bottom="4px" top="4px"></font_margin>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.DefaultFontMarginBottom = "4px";
        DEXT5.config.DefaultFontMarginTop = "4px";

        new Dext5editor("editor1");

    </script>       
</div>
