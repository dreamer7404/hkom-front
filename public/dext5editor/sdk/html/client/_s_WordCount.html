DEXT5 Editor :: WordCount

XML : <word_count>.....</word_count>, Config : config.WordCount.Use
XML : <word_count count_white_space="....."></word_count>, Config : config.WordCount.CountWhiteSpace
XML : <word_count limit="....."></word_count>, Config : config.WordCount.Limit
XML : <word_count limit_byte="....."></word_count>, Config : config.WordCount.LimitByte
XML : <word_count limit_char="....."></word_count>, Config : config.WordCount.LimitChar
XML : <word_count limit_line="....."></word_count>, Config : config.WordCount.LimitLine
XML : <word_count limit_message="....."></word_count>, Config : config.WordCount.LimitMessage

Config 설정이 우선 적용됩니다.
에디터 우측 하단에 글자수와 행의 표시유무 및
글자수 제한, 공백포함 글자수, alert 출력여부 등을 설정합니다.

remarks

word_count  기본값은 "0"으로 미표시, "1"로 설정시 글자수와 행을 우측 하단에 표시합니다.
count_white_space  문자수에 공백을 포함할지 설정합니다. (0 : 포함안함, 1 : 포함)
limit  글자수의 제한여부를 설정합니다. (0 : 제한안함, 1 : 제한함)
limit_byte  제한할 글자의 Byte를 설정(숫자입력)합니다. (limit_char의 값이 설정되어있으면 limit_char의 설정을 우선으로 합니다.)
limit_char  제한할 글자수를 설정(숫자입력)합니다.
limit_line  입력 제한할 행의 수를 설정(숫자입력)합니다.
limit_message  글자수 또는 행의 제한 초과시 alert의 출력 여부를 설정합니다. (0 : 출력안함, 1 : 출력 / WordCount.Limit="1" 일때 작동합니다.)

sample code

에디터 우측 하단에 글자수와 행의 표시유무 및 글자수 제한, 공백포함 글자수, alert 출력여부 등을 설정합니다.

XML

.....
<setting>
    <word_count count_white_space="1" limit="1" limit_char="4000" limit_line="100" limit_message="1">1</word_count>
    <-- Byte 단위로 제한하도록 설정할 시 limit_char 대신 limit_byte를 설정하시면 됩니다. -->
    .....
</setting>
.....
