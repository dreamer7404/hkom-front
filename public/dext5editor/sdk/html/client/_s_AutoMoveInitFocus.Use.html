DEXT5 Editor :: AutoMoveInitFocus.Use

XML : <auto_move_init_focus>.....</auto_move_init_focus>, Config : config.AutoMoveInitFocus.Use

Config 설정이 우선 적용됩니다.
에디터 생성 후 자동으로 원래 포커스를 가지고 있던 객체에 포커스를 이동시켜 주도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 으로 설정 시 에디터 생성 후 자동으로 원래 포커스를 가지고 있던 객체에 포커스를 이동시켜 주도록 설정합니다.

sample code

에디터 생성 후 자동으로 원래 포커스를 가지고 있던 객체에 포커스를 이동시켜 주도록 설정합니다.

XML

.....
<setting>
    <auto_move_init_focus>1</auto_move_init_focus>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
