[IE, FireFox 브라우저 전용]

DEXT5 Editor :: AfterPasteCaretPosition

XML : <after_paste_caret_position>.....</after_paste_caret_position>
Config : config.AfterPasteCaretPosition

Config 설정이 우선 적용됩니다.
붙여넣기 후 커서 위치가 문단의 처음 또는 끝으로 이동하도록 설정합니다.

remarks

기본값은 "0" 이며 붙여넣기 후 커서 위치가 문단의 처음 이동하도록 설정,
"1" 으로 설정 시 붙여넣기 후 커서 위치가 문단의 끝으로 이동하도록 설정합니다.

sample code

IE, FireFox 브라우저 사용 시 붙여넣기 후 커서 위치가 문단의 끝으로 이동하도록 설정합니다.

XML

.....
<setting>
    <after_paste_caret_position>1</after_paste_caret_position>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AfterPasteCaretPosition = "1";

        new Dext5editor("editor1");

    </script>       
</div>
