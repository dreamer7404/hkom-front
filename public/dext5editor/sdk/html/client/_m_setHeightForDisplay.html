DEXT5 Editor :: setHeightForDisplay

void setHeightForDisplay(editor_id)

에디터의 작성한 내용보다 에디터의 높이가 짧으면(스크롤이 있고, 잘린 내용이 있으면) 한 번에 볼 수 있도록 크기를 맞춰서 늘립니다.

return value

없음.

parameters

editor_id  에디터의 id를 의미합니다.

remarks

에디터 설정파일(dext5editor/config/dext_editor.xml)의 view_mode_auto_width 혹은 view_mode_auto_height 설정값이 "1" 이고,
mode가 "view"로 설정 시 동작합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5ForbiddenWord(){
        // id가 editor1인 에디터의 작성한 내용보다 에디터의 높이가 짧으면 한번에 볼 수 있도록 크기를 맞춰서 늘립니다.
        DEXT5.setHeightForDisplay('editor1');
        // work something ...
    }

</script>
