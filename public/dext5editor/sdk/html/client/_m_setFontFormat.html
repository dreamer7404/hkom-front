DEXT5 Editor :: setFontFormat

void setFontFormat(formatType, editor_id)

에디터 body영역 안의 select된 부분의 서식을 파라미터의 서식으로 적용/해제합니다.

return value

없음.

parameters

formatType  변경할 포맷스타일을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

formatType에는 다음과 같은 값들이 올수 있습니다.
"bold", "underline", "italic", "strike_through", "superscript", "subscript", "remove_format"

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>

    function dext5Set(){
        var format = "bold";
        // id가 editor1인 에디터 body영역 안의 select된 부분의 font format중 bold를 적용/해제합니다.
        DEXT5.setFontFormat(format, 'editor1');
        // work something ...
    }

</script>
