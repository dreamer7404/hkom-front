DEXT5 Editor :: XssAllowUrl

XML : <xss_allow_url>.....</xss_allow_url>, Config : config.XssAllowUrl

Config 설정이 우선 적용됩니다.
XSS 방지하지 않을(허용할) url을 설정합니다.

remarks

해당 URL을 src에 포함한 태그는 XSS를 혀용합니다.
script에서 config 적용시 URL(http, https ~ )을 콤마(,)로 구분하여 입력합니다.

sample code

XSS 방지하지 않을(허용할) url을 설정합니다.

XML

.....
<xss_protection>
    .....
    <xss_allow_url>
        <allow_url>.....</allow_url>
        <allow_url>.....</allow_url>
    </xss_allow_url>
    .....
</xss_protection>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
