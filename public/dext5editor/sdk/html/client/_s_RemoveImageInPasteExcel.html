DEXT5 Editor :: RemoveImageInPasteExcel

XML : <remove_image_in_paste_excel>.....</remove_image_in_paste_excel>,
Config : config.RemoveImageInPasteExcel

Config 설정이 우선 적용됩니다.
엑셀 내용 붙여넣기 시 이미지를 제거하도록 설정합니다.

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 엑셀 내용 붙여넣기 시 이미지를 제거합니다.
클립보드에 접근 가능한 경우에만 적용이 가능합니다.

sample code

엑셀 내용 붙여넣기 시 이미지를 제거합니다.

XML

.....
<setting>
    <remove_image_in_paste_excel>1</remove_image_in_paste_excel>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">
