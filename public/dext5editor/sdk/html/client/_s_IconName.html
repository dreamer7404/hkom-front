DEXT5 Editor :: IconName

XML : <icon_name>.....</icon_name>, Config : config.IconName

Config 설정이 우선 적용됩니다.
설정한 값의 editor_"icon_name".css 파일을 불러오고 icon image url의 파일명도 해당 icon_name으로 불러옵니다.

remarks

기본값은 "icon" 이며, 불러오는 에디터의 아이콘 css 파일명과 아이콘 이미지 url을 별도로 설정할 경우 사용됩니다.

sample code

에디터의 아이콘 css 파일명및 아이콘 이미지 url을 editor_newIconCSS.css로 설정합니다.

XML

.....
<setting>
    <icon_name>newIconCSS</icon_name>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.IconName = "newIconCSS";

        new Dext5editor("editor1");

    </script>       
</div>
