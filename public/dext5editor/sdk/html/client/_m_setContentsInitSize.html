DEXT5 Editor :: setContentsInitSize

void setContentsInitSize(pContentInfo, editor_id)

pContentInfo에 담긴 contents들을 에디터 변경의 기준점(초기값)으로 설정합니다. (이미지갯수, 이미지총용량)

return value

없음.

parameters

pContentInfo  초기화로 지정할 컨텐츠의 리스트를 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

초기값은 contents 추가 등 이전 상황과 비교할때의 기준값으로 활용됩니다.
