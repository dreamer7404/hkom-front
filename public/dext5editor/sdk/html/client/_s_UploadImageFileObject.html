DEXT5 Editor :: UploadImageFileObject

XML : <uploader_setting upload_image_file_object="....."></uploader_setting>
Config : config.UploadImageFileObject

Config 설정이 우선 적용됩니다.
붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 file object로 업로드 할지 설정합니다.

remarks

기본값은 "0" 이며 붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 file object로 업로드하지 않고,
"1" 로 설정하는 경우, 붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 file object로 업로드 합니다.

sample code

붙여넣기 또는 드래그앤 드랍으로 이미지 삽입 시 file object로 업로드 합니다.

XML

.....
<uploader_setting>
    .....
    <uploader_setting upload_image_file_object="1"></uploader_setting>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
