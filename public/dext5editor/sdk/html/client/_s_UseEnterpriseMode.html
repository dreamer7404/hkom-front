DEXT5 Editor :: UseEnterpriseMode

XML : <use_enterprise_mode>.....</use_enterprise_mode>, Config : config.UseEnterpriseMode

Config 설정이 우선 적용됩니다.
엔터프라이즈 모드를 사용하는 환경일 시 해당 환경임을 알려주도록 설정합니다.

remarks

"0"으로 설정 시 설정안함, "1"로 설정 시 엔터프라이즈 모드를 사용하는 환경일 시 해당 환경임을 알려줍니다.
기본값은 "0" 입니다.

sample code

엔터프라이즈 모드를 사용하는 환경일 시 해당 환경임을 알려주도록 설정합니다.

XML

.....
<setting>
    <use_enterprise_mode>1</use_enterprise_mode>
    .....
</setting>
.....
