DEXT5 Editor :: OpenDocument.PowerPoint

XML : <powerpoint max_slide="....."></powerpoint>, Config : config.OpenDocument.PowerPoint.MaxSlide
XML : <powerpoint max_size="....."></powerpoint>, Config : config.OpenDocument.PowerPoint.MaxSize

Config 설정이 우선 적용됩니다.
파워포인트파일의 열기 관련 설정을 적용합니다.

remarks

max_slide  파일 열기할 powerpoint파일의 최대 slide 수를 설정합니다.
max_size  파일 열기할 powerpoint파일의 최대 용량을 설정합니다.

sample code

파워포인트파일의 열기 관련 설정을 적용합니다.

XML

.....
<setting>
    .....
    <open_document>
        <powerpoint max_slide="2", max_size="10000"></powerpoint>
    </open_document>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.OpenDocument.PowerPoint = {
            MaxSlide : "2",
            MaxSize : "10000",
        }

        new Dext5editor("editor1");

    </script>       
</div>
