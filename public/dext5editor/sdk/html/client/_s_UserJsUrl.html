DEXT5 Editor :: UserJsUrl

XML : <user_js_url>.....</user_js_url>, Config : config.UserJsUrl

Config 설정이 우선 적용됩니다.
에디터 로드시 사용자의 js를 에디터 body의 head부분에 import 해주도록 설정합니다.

remarks

import할 사용자의 js url(전체)을 입력합니다.
상대경로도 입력이 가능합니다.

sample code

에디터 로드시 사용자의 js를 에디터 body의 head부분에 import 합니다.

XML

.....
<setting>
    <user_js_url>.....</user_js_url>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UserJsUrl = ".....";

        new Dext5editor("editor1");

    </script>       
</div>
