DEXT5 Editor :: getFileSizeEx

void getFileSizeEx(fileSizePath, callBackFunc, editor_id)

파일경로를 배열로 받아 해당 배열 경로들의 각 파일 크기를 콤마(,)로 구분한 String값으로 callBackFunc의 첫번째 파라미터로 반환해줍니다.

return value

없음.

parameters

fileSizePath  파일 경로정보들을 포함한 배열을 의미합니다.
callBackFunc  실행시킬 callback function을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

callback function의 첫번째 파라미터로 각 파일 크기를 콤마(,)로 구분한 String값이 반환됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5GetSize(){
        // 파일 경로들을 포함한 배열을 생성합니다.
        var tempArr = ["D:\\temp1.txt","D:\\temp2.txt"];
        // callback function을 설정합니다.
        var cbf = function(filesize){
            console.log(filesize);
        }
        // tempArr내의 경로들에 해당하는 각 파일 크기를 콤마(,)로 구분한 String값을 callback function의 첫번째 파라미터로 반환받습니다.
        DEXT5.getFileSizeEx(tempArr, cbf, 'editor1');
        // work something ...
    }

</script>
