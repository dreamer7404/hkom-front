[IE 브라우저 전용]

DEXT5 Editor :: IeBugFixedApplyAllBrowser

XML : <apply_all_browser>.....</apply_all_browser>, Config : config.IeBugFixedApplyAllBrowser

Config 설정이 우선 적용됩니다.
테이블 align 제거와 관련하여 IE브라우저 및 타 브라우저가 동일하게 반영되도록 설정합니다.

remarks

기본값은 "0" 이고 반영 안함,
"1"로 설정 시 반영합니다.

sample code

테이블 align 제거와 관련하여 IE브라우저 및 타 브라우저가 동일하게 반영되도록 설정합니다.

XML

.....
<setting>
    .....
    <browser_bugfixed>
        <apply_all_browser>1</apply_all_browser>
    </browser_bugfixed>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.IeBugFixedApplyAllBrowser = "1";

        new Dext5editor("editor1");

    </script>       
</div>
