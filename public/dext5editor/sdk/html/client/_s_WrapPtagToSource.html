DEXT5 Editor :: WrapPtagToSource

XML : <wrap_ptag_to_source>.....</wrap_ptag_to_source>
Config : config.WrapPtagToSource
XML : <wrap_ptag_to_source apply_to_get_api="....."></wrap_ptag_to_source>
Config : config.WrapPtagToGetApi
XML : <wrap_ptag_to_source skip_tag="....."></wrap_ptag_to_source>
Config : config.WrapPtagSkipTag

Config 설정이 우선 적용됩니다.
set API가 실행될 때 block 태그를 제외한 태그는 p 태그로 감싸는 설정입니다.

remarks

wrap_ptag_to_source  기본값은 "0" 이고, "1" 로 설정 시 p 태그로 감싸며, "2"로 설정 시 내용이 없는 textNode는 제외하고 감싸도록 설정합니다.
apply_to_get_api  get API 사용시 태그가 없는 부분을 p 태그로 감싸도록 설정합니다. (0 : 설정안함, 1 : 설정함)
skip_tag  p 태그로 감싸는 로직 수행시 제외할 태그를 설정합니다. (태그명을 입력하여 설정합니다.)

sample code

set API 및 get API가 실행될 때 block 태그를 제외한 태그는 ptag로 감싸도록 설정합니다.

XML

.....
<setting>
    .....
    <wrap_ptag_to_source apply_to_get_api="1" skip_tag=".....">1</wrap_ptag_to_source>
    .....
</setting>
.....
