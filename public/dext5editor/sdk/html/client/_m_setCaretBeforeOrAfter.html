DEXT5 Editor :: setCaretBeforeOrAfter

void setCaretBeforeOrAfter(option, editor_id)

에디터 작성영역에서 특정 노드를 앞/뒤 쪽에 커서를 위치하도록 합니다.

return value

없음.

parameters

option  API에 적용할 option을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

option.node : 작성영역에 존재하는 Node
option.pos : 커서를 놓을 위치 ("before"||"after")

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Set(){
        // 테스트용 현재 위치한 커서의 node를 가져옵니다.
        var tempElem = DEXT5.getCaretElement();
        var option = {
            node:tempElem,
            pos:"before"
        }
        // id가 editor1인 에디터에 option값으로 setCaretBeforeOrAfter API를 수행합니다.
        DEXT5.setCaretBeforeOrAfter(option, 'editor1');
        // work something ...
    }

</script>
