[플러그인 전용]

DEXT5 Editor :: AllowOpenFileType

XML : <allow_open_file_type>.....</allow_open_file_type>, Config : config.AllowOpenFileType

Config 설정이 우선 적용됩니다.
플러그인 모드의 문서열기시 허용할 사용자정의 확장자를 설정합니다.

remarks

확장자를 콤마(,)로 구분하여 입력합니다.

sample code

플러그인 모드의 문서열기시 hwp, doc, txt, pdf 확장자의 열기를 허용합니다.

XML

.....
<setting>
    <allow_open_file_type>hwp,doc,txt,pdf</allow_open_file_type>
    .....
</setting>
.....
