DEXT5 Editor :: ZoomList

XML : <zoom>.....</zoom>, Config : config.ZoomList

Config 설정이 우선 적용됩니다.
우측 하단의 확대/축소 설정 콤보박스에 표시할 비율을 설정합니다.

remarks

script에서 config으로 설정시 확대/축소 비율을 콤마(,)로 구분하여 입력합니다.

sample code

확대/축소 설정 콤보박스에 표시할 비율을 설정합니다.

XML

.....
<zoom>
    .....
    <item>50%</item>
    <item>75%</item>
    <item>100%</item>
    <item>125%</item>
    <item>150%</item>
    <item>175%</item>
    <item>200%</item>
    .....
</zoom>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ZoomList = "50%,75%,100%,125%,150%,175%,200%";

        new Dext5editor("editor1");
    
    </script>       
</div>
