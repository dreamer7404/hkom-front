DEXT5 Editor :: setUserFontSize

void setUserFontSize(_fontSize, editor_id)

에디터의 기본 폰트사이즈 및 에디터에 작성된 폰트사이즈를 해당 사이즈로 변경합니다.

return value

없음.

parameters

_fontSize  설정할 폰트사이즈를 의미합니다. (숫자만 입력합니다.)
editor_id  에디터의 id를 의미합니다.

remarks

없음.
