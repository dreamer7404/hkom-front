DEXT5 Editor :: OpenDocument.Excel

XML : <excel max_sheet="....."></excel>, Config : config.OpenDocument.Excel.MaxSheet
XML : <excel max_size="....."></excel>, Config : config.OpenDocument.Excel.MaxSize

Config 설정이 우선 적용됩니다.
엑셀파일의 열기 관련 설정을 적용합니다.

remarks

max_sheet  파일 열기할 excel파일의 최대 sheet 수를 설정합니다.
max_size  파일 열기할 excel파일의 최대 용량을 설정합니다.

sample code

엑셀파일의 열기 관련 설정을 적용합니다.

XML

.....
<setting>
    .....
    <open_document>
        <excel max_sheet="2", max_size="10000"></excel>
    </open_document>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.OpenDocument.Excel = {
            MaxSheet : "2",
            MaxSize : "10000",
        }

        new Dext5editor("editor1");

    </script>       
</div>
