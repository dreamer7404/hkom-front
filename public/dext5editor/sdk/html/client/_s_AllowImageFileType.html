DEXT5 Editor :: AllowImageFileType

XML : <allow_image_file_type>.....</allow_image_file_type>
Config : config.AllowImageFileType
XML : <allow_image_file_type max_file_size="....."></allow_image_file_type>
Config : config.MaxImageFileSize
XML : <allow_image_file_type max_base64file_count="....."></allow_image_file_type>
Config : config.MaxImageBase64fileCount

Config 설정이 우선 적용됩니다.
이미지 삽입 시 허용할 확장자를 설정합니다.
이미지 삽입 시 허용할 최대 파일 사이즈를 제한합니다.
이미지 삽입 시 허용할 최대 Base64 이미지 파일 개수를 제한합니다.

remarks

allow_image_file_type   확장자를 ","(콤마)로 구분하여 입력합니다.
max_file_size   용량을 바이트단위까지 입력합니다.
max_base64file_count   제한할 개수를 입력합니다.

sample code

이미지 삽입 시 허용할 확장자를 "jpg,png"로 설정합니다.
이미지 삽입 시 허용할 최대 파일 사이즈를 "5MB"로 설정합니다.
이미지 삽입 시 허용할 최대 Base64 이미지 파일 개수를 3개로 설정합니다.

XML
.....
<uploader_setting>
    .....
    <allow_image_file_type max_file_size="5MB" max_base64file_count="3">jpg,png</allow_image_file_type>
    .....
</uploader_setting>
.....
