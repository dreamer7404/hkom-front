DEXT5 Editor :: LetterSpacingList

XML : <letter_spacing>.....</letter_spacing>, Config : config.LetterSpacingList

Config 설정이 우선 적용됩니다.
에디터 글자간격 메뉴에 노출할 크기 리스트를 설정합니다.

remarks

XML:letter_spacing  필요 시 <spacing>.....</spacing> 항목을 추가하여 설정합니다.
Config:LetterSpacingList  필요 시 DEXT5.config.LetterSpacingList = "Spacing값";에 콤마(,)로 구분하여 설정합니다.

sample code

에디터 글자간격 메뉴에 노출할 크기를 0.5pt, 1pt, 1.5pt 로 설정합니다.

XML

.....
<letter_spacing>
    <spacing>0.5</spacing>
    <spacing>1</spacing>
    <spacing>1.5</spacing>
</letter_spacing>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.LetterSpacingList = "0.5pt,1pt,1.5pt";

        new Dext5editor("editor1");

    </script>       
</div>
