DEXT5 Editor :: TabIndexObjValue

config.TabIndexObjValue

Config 설정이 우선 적용됩니다.
에디터 로드 후 tabIndexObj로 설정된 id를 가진 객체의 tabindex를 설정할 값을 받습니다.

remarks

tabIndexObj로 설정된 id를 가진 객체가 존재할 시 적용됩니다.
필요 시 DEXT5.config.TabIndexObjValue = "100" 으로 설정합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div border="0" style="width:900px;height:550px">    
    <script type="text/javascript">

        // tabIndexObj로 설정된 id를 가진 객체의 tabindex를 -1 으로 설정합니다.
        DEXT5.config.TabIndexObjValue = "-1";

        new Dext5editor("editor1");

    </script>       
</div>
