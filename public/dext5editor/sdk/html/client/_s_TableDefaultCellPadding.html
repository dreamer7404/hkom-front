DEXT5 Editor :: TableDefaultCellPadding

XML : <table_property cell_padding="....."></table_property>, Config : config.TableDefaultCellPadding

Config 설정이 우선 적용됩니다.
표 생성 시 셀 안쪽의 기본 여백(cellpadding)을 설정합니다.

remarks

여백(cellpadding)으로 설정할 숫자를 입력합니다.

sample code

.....
<setting>
    .....
    <table_property cell_padding="50">
        .....
    </table_property>
    .....
