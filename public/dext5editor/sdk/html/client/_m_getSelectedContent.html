DEXT5 Editor :: getSelectedContent

Object getSelectedContent(editor_id)

에디터의 작성영역에 선택된 부분의 text와 html을 Object로 반환해줍니다.

return value

에디터 작성영역에 선택된 부분의 text와 html 정보를 담은 Object

parameters

editor_id  에디터의 id를 의미합니다.

remarks

return은 { text : '선택된 부분의 text' , html : '선택된 부분의 html' } 형태로 반환 합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>

        var dext5 = DEXT5.getSelectedContent('editor1');
        alert(dext5.text);
        alert(dext5.html);
        // work something ...
    }
