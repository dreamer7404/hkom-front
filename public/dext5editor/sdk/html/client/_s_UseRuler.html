Config 설정이 우선 적용됩니다.
XML:use_ruler,Config:Use  가로 눈금자 표시 사용여부를 설정합니다.
XML:ruler_init_position,Config:InitPosition  눈금자 포인터의 위치 값(픽셀)을 설정합니다.
XML:view_pointer,Config:ViewPointer  눈금자 포인터 아이콘 표시 사용여부를 설정합니다.
XML:view_guide_line,Config:ViewGuideLine  에디터 작성 영역에 눈금자 포인터 기준선(세로) 표시 사용여부를 설정합니다.
XML:view_ruler,Config:ViewRuler  에디터 작성 영역 상단에 눈금자 이미지 표시 사용여부를 설정합니다.
XML:guide_line_color,Config:GuideLineColor  눈금자 포인터 아이콘 아래의 세로선 색상을 설정합니다.
XML:guide_line_style,Config:GuideLineStyle  눈금자 포인터 아이콘 아래의 세로선 CSS style을 설정합니다.
XML:default_view,Config:DefaultView  에디터 생성후 로드될때 최초 눈금자의 표시여부를 설정합니다.
XML:auto_fit_mode,Config:AutoFitMode  에디터 크기에 맞게 body 크기를 동적으로 맞춰주도록 설정합니다.
XML:fix_editor_width,Config:FixEditorWidth  AutoFitMode가 "1"(사용)일 때에 에디터 body의 style에서 width 항목의 제거 여부를 설정합니다.
XML:use_vertical,Config:UseVertical  세로 눈금자 표시 사용여부를 설정합니다.

remarks

XML:use_ruler,Config:Use  기본값은 "0" 이고 눈금자 사용 안함, "1"로 설정 시 눈금자 사용으로 설정합니다.
XML:ruler_init_position,Config:InitPosition  기본값은 "680px" 입니다.
XML:view_pointer,Config:ViewPointer  기본값은 "1" 이고 표시, "0" 으로 설정 시 포인트 아이콘을 표시 안함으로 설정합니다.
XML:view_guide_line,Config:ViewGuideLine  기본값은 "1" 이고 표시, "0" 으로 설정 시 눈금자 포인트 기준선 표시 안함으로 설정합니다.
XML:view_ruler,Config:ViewRuler  기본값은 "1" 이고 표시, "0" 으로 설정 시 눈금자 이미지 표시 안함으로 설정합니다.
XML:guide_line_color,Config:GuideLineColor  기본값은 rgb(255, 0, 0) 입니다. (색상의 HEX, rgb 등으로 설정 가능합니다.)
XML:guide_line_style,Config:GuideLineStyle  기본값은 dotted 입니다. (border-style css값으로 설정 가능합니다.)
XML:default_view,Config:DefaultView  기본값은 "1" 이고 에디터 로드시 최초 눈금자 표시, "0" 으로 설정 시 최초 눈금자 표시 안함으로 설정합니다.
XML:auto_fit_mode,Config:AutoFitMode  기본값은 "0" 이고 사용안함, "1" 로 설정시 에디터 크기에 맞게 body 크기를 동적으로 맞추도록 설정합니다.
XML:fix_editor_width,Config:FixEditorWidth  기본값은 "1" 이고 에디터 body의 style에서 width 항목을 제거 안함,
"0" 으로 설정시 에디터 body의 style에서 width 항목을 제거 합니다. (AutoFitMode가 "1" 일때 적용됩니다.)
XML:use_vertical,Config:UseVertical  기본값은 "0" 이고 세로 눈금자 사용 안함, "1"로 설정 시 세로 눈금자 사용으로 설정합니다.
