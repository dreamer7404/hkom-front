DEXT5 Editor :: RemoveEmptyTag

XML : <remove_empty_tag>.....</remove_empty_tag>
Config : config.RemoveEmptyTag
XML : <remove_empty_tag insert_nbsp_for_line_break="....."></remove_empty_tag>
Config : config.RemoveEmptyTagInsertNbspForLineBreak
XML : <remove_empty_tag replace_default_paragraph_value="....."></remove_empty_tag>
Config : config.ReplaceDefaultParagraphValue
XML : <remove_empty_tag use_set_value="....."></remove_empty_tag>
Config : config.RemoveEmptyTagSetValue

Config 설정이 우선 적용됩니다.
get API 사용 시 내용이 없는 빈 태그 삭제 여부을 적용합니다.
설정값에 따라서 set API 사용 시에도 적용합니다.

remarks

remove_empty_tag  기본값은 "1"이며 get API 사용시 빈 태그 삭제함, "0"으로 설정 시 삭제하지 않습니다.
insert_nbsp_for_line_break  기본값은 "1"이며 RemoveEmptyTag 옵션을 사용 시 삭제 대신 태그 안에 &nbsp;를 넣어주도록 설정합니다.
replace_default_paragraph_value  기본값은 "1"이며 빈 p태그에 기본 값을 넣지 않도록 할 수 있게 설정합니다.
use_set_value  RemoveEmptyTag 옵션을 set API 사용 시에도 적용할지에 대한 옵션을 설정합니다.

sample code

빈 태그 삭제 관련 설정들을 적용합니다.

XML

.....
<setting>
    <remove_empty_tag insert_nbsp_for_line_break="1", replace_default_paragraph_value="1" use_set_value="1">1</remove_empty_tag>
    .....
</setting>
.....
