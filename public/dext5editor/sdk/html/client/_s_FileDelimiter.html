DEXT5 Editor :: FileDelimiter

XML : <file_delimiter unit_attribute_delimiter="....."></file_delimiter>, Config : config.UnitAttributeDelimiter
XML : <file_delimiter unit_delimiter="....."></file_delimiter>, Config : config.UnitDelimiter

Config 설정이 우선 적용됩니다.
에디터에서 img,embed,a,video 등의 src를 가져올 때 구분자를 설정합니다.

remarks

XML:unit_attribute_delimiter,Config:UnitAttributeDelimiter  src와 file name 사이의 구분문자로 설정합니다.
XML:unit_delimiter,Config:UnitDelimiter   각 src 사이의 구분자를 설정합니다.

sample code

입력값을 유니코드값으로 하여 변환된 문자열을 구분자로 설정합니다.

XML

.....
<setting>
    <file_delimiter unit_attribute_delimiter="11" unit_delimiter="12"></file_delimiter>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UnitAttributeDelimiter = "11";
        DEXT5.config.UnitDelimiter = "12";

        new Dext5editor("editor1");

    </script>       
</div>
