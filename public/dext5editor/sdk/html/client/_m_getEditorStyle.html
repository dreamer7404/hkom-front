DEXT5 Editor :: getEditorStyle

String getEditorStyle(element, propertyName, editor_id)

파라미터로 넘겨준 element의 스타일 정보를 반환해줍니다.

return value

해당 element의 스타일 값을 Object 형태로 반환합니다.

parameters

element  style 정보를 참조할 element를 의미합니다.
propertyName  반환받을 style의 property명을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

propertyName 파라미터에 "all" 입력시 style 정보를 전부 포함한 Object를 반환받습니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>

    function dext5Style(){
        // 테스트용 element(커서위치)를 id가 editor1인 에디터에서 가져옵니다.
        var tempElement = DEXT5.getCaretElementEx('editor1');
        // 해당 element의 color style 정보를 반환받습니다.
        var colorStyle = DEXT5.getEditorStyle(tempElement,'color','editor1');
        alert(colorStyle);
        // work something ...
    }

</script>
