DEXT5 Editor :: HtmlprocessCustomText

XML : <htmlprocess_custom_text>.....</htmlprocess_custom_text>, Config : config.HtmlprocessCustomText

Config 설정이 우선 적용됩니다.
에디터의 프로세스 로딩 문구를 설정합니다.

remarks

기본값은 "<b>Processing...</b>" 이며,
문구 설정시 에디터의 프로세스 로딩 문구로 적용됩니다.
XML 설정시 문구의 Style을 지정할 수 없습니다.

sample code

에디터의 프로세스 로딩 문구를 <b style="color:darkcyan">작업중...</b>으로 설정합니다.
XML 설정시에는 문구의 Style을 제외한 텍스트만 지정합니다.

XML

.....
<setting>
    <htmlprocess_custom_text>작업중...</htmlprocess_custom_text>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.HtmlprocessCustomText = "<b style="color:darkcyan">작업중...</b>";

        new Dext5editor("editor1");

    </script>       
</div>
