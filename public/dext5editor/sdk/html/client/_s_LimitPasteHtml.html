DEXT5 Editor :: LimitPasteHtml

XML : <limit_paste_html_length>.....</limit_paste_html_length>
Config : config.LimitPasteHtml
XML : <limit_paste_html_length length="....."></limit_paste_html_length>
Config : config.LimitPasteHtmlLength

Config 설정이 우선 적용됩니다.
붙여넣기시 내용이 많을 경우를 체크할 때 사용합니다.
엑셀, ppt등 에서 붙여넣을 때 HTML로 붙여넣기시 설정한 크기에 따라 붙여넣기를 비활성화 여부를 설정합니다.

remarks

limit_paste_html_length  엑셀, ppt등 에서 붙여넣을 때 HTML로 붙여넣기시 설정한 크기에 따라 붙여넣기를 비활성화 여부를 설정합니다.
limit_paste_html_length@length  붙여넣기가 비활성화 될 기준(크기)을 설정합니다.

sample code

엑셀, ppt등 에서 붙여넣을 때 HTML로 붙여넣기시 길이가 500을 넘으면 붙여넣기를 비활성화 합니다.

XML

.....
<setting>
    .....
    <limit_paste_html_length length="500">1</limit_paste_html_length>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->

        DEXT5.config.LimitPasteHtmlLength = "500";
