DEXT5 Editor :: CleanNestedTagOptions

XML : <remove_tag>.....</remove_tag>, Config : config.CleanNestedTagOptions.RemoveTag
XML : <nested_count>.....</remove_tag>, Config : config.CleanNestedTagOptions.NestedCount

Script 설정이 우선 적용 됩니다.
중복 태그 제거 사용 시 태그 정리 기능을 설정합니다.

remarks

remove_tag   제거할 태그명을 콤마(,)로 구분하여 입력합니다.
nested_count   기본값은 "50" 이며 중복된 태그 제거 기준 개수를 설정합니다.

sample code

p,div 태그를 30개 이상 중복된 태그가 있을 경우 제거하도록 설정합니다.

XML

.....
<setting>
    .....
    <clean_nested_tag_options>
        <remove_tag>p,div</remove_tag>
        <nested_count>30</nested_count>
        .....
    </clean_nested_tag_options>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.CleanNestedTagOptions.RemoveTag = "p,div";
        DEXT5.config.CleanNestedTagOptions.NestedCount = "30";

        new Dext5editor("editor1");

    </script>       
</div>
