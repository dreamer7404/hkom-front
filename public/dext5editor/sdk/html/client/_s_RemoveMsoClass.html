DEXT5 Editor :: RemoveMsoClass

XML : <remove_mso_class>.....</remove_mso_class>, Config : config.RemoveMsoClass

Config 설정이 우선 적용됩니다.
set API사용 또는 붙여넣기 시 Mso로 시작하는 클래스를 지울지에 대하여 설정합니다.

remarks

기본값은 "1"로 set 또는 붙여넣기 시 Mso로 시작하는 클래스를 제거함,
"0"으로 설정 시 제거하지 않습니다.
제거하고 싶지 않은 Mso로 시작하는 클래스가 존재할 경우 활용 가능합니다.

sample code

set API사용 또는 붙여넣기 시 Mso로 시작하는 클래스를 제거하지 않습니다.

XML

.....
<setting>
    <remove_mso_class>0</remove_mso_class>
    .....
</setting>
.....
