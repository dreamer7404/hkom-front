DEXT5 Editor :: dext_input_event

void dext_input_event(inputStr, eventObject, currElem, editorId)

에디터 body에 conetent를 붙여넣기 하거나 키보드 입력으로 텍스트를 입력, 에디터 body의 작성내용이 지워질 때마다 발생합니다.

return value

없음.

parameters

inputStr  "input"의 String값 입니다.
eventObject  data, inputType, path, target, timeStamp등의 정보를 포함한 InputEvent 객체 입니다.
currElem  이벤트가 발생한 에디터에서 현재 커서 위치의 객체 입니다.
editorId  이벤트가 발생한 에디터의 id입니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 return_event 설정의 input_event 속성값이 1인 경우에 이벤트가 발생합니다.
