DEXT5 Editor :: getForbiddenWordValidation

Boolean getForbiddenWordValidation(editor_id)

에디터의 작성영역에 금지어가 존재하는지 여부를 확인합니다.

return value

에디터의 작성영역에 금지어가 존재하는지 여부

parameters

editor_id  에디터의 id를 의미합니다.

remarks

작성 영역에 금지어가 있거나, 설정된 금지어가 없을경우 true를 반환합니다.
작성 영역에 금지어가 없을경우 false를 반환합니다.
금지어 설정은 에디터 설정파일(dext5editor/config/dext_editor.xml)의 forbidden_word 을 확인하시면 됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5IsForbiddenWord(){
        // id가 'editor1'인 에디터의 작성영역에 금지어가 존재하는지 여부를 확인합니다.
        var isForbidden = DEXT5.getForbiddenWordValidation('editor1');
        alert(isForbidden);
        // work something ...
    }

</script>

    </script>
