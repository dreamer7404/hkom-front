DEXT5 Editor :: RemoveLangAttribute

XML : <remove_lang_attribute>.....</remove_lang_attribute>, Config : config.RemoveLangAttribute

Config 설정이 우선 적용됩니다.
set API 사용시 태그안에 있는 lang attribute를 제거하도록 설정합니다.

remarks

기본값은 "1"로 set API 사용시 태그안에 있는 lang attribute를 제거합니다.
"0"으로 설정 시 사용하지 않습니다.
Office 프로그램에서 내용을 붙여넣기 할 때 속도 문제가 발생할 경우 활용 가능합니다.

sample code

set API 사용시 태그안에 있는 lang attribute를 제거합니다.

XML

.....
<setting>
    <remove_lang_attribute>1</remove_lang_attribute>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.RemoveLangAttribute = "1";

        new Dext5editor("editor1");

    </script>       
</div>
