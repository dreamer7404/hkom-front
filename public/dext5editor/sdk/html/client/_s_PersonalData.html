DEXT5 Editor :: PersonalData

XML : <personal_data>.....</personal_data>, Config : config.PersonalData

Config 설정이 우선 적용됩니다.
개인정보 추출 시 주민번호, 이메일, 전화번호를 검색할 수 있고, 추가로 custom하여 검색할 수 있습니다.

remarks

기본값은 "email,phone,RRN" 이며 각각 이메일, 전화번호, 주민등록번호를 추출할 항목으로 설정합니다.
추가로 custom 으로 설정 시 config/dext_personal_data.js 에 정의된 내용으로 동작하게 됩니다.
해당 파일에 기본적으로
이메일(email), 전화번호(phone), 주민등록번호(rrn), 여권번호(passport), 운전면허번호(driver), 외국인번호(foreigner)가 정의되어 있습니다.

sample code

이메일, 전화번호, 주민등록번호를 개인정보로 추출하도록 설정합니다.

XML

.....
<setting>
    <personal_data>email,phone,RRN</personal_data>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PersonalData = "email,phone,RRN";

        new Dext5editor("editor1");

    </script>       
</div>
