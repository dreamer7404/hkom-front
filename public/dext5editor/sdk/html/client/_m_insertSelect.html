DEXT5 Editor :: insertSelect

void insertSelect(attributesObj, optionsTextArr, optionsValueArr, optionsSelectedArr, selectObj, editor_id)

에디터 선택영역에 attributesObj의 정보로 select 객체를 생성하여 삽입합니다.
selectObj 오브젝트가 에디터의 body에 존재하면 해당 select 객체를 attributesObj의 정보로 변경합니다.

return value

없음.

parameters

attributesObj  생성 또는 교체할 seleect 객체의 정보를 의미합니다.
optionsTextArr  select option으로 추가할 text 배열을 의미합니다.
optionsValueArr  추가할 각 option의 value 배열을 의미합니다.
optionsSelectedArr  selected 할 option 배열을 의미합니다. (해당 index의 값이 ""(공백)이 아니면 check 됩니다.)
selectObj  에디터 body에 존재하는 교체대상의 select 객체를 의미합니다. (생성을 원하면 null 입력)
editor_id  에디터의 id를 의미합니다.

remarks

attributesObj는 다음과 같은 정보를 설정합니다.
name, id, class, title, multiple, size, disabled, style.width, style.height

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Input(){
        // 테스트용 select 생성정보
        var Obj = {
            name:"DEXT5", 
            id:"DEXT5EDITOR", 
            class:"DEXT5CLASS", 
            title:"DEXT5TITLE",
            multiple:true,
            size:"20",
            disabled:false, 
            "style.width":"200px", 
            "style.height":"100px"
        };
        // 1, 2, 3을 select option으로 만듭니다.
        var optionsTextArr = ['1', '2', '3'];
        // 1, 2, 3 option에 각각 'one', 'two', 'three'의 값을 value로 지정합니다.
        var optionsValueArr = ['one', 'two', 'three'];
        // 1, 3 option을 선택상태로 만듭니다.
        var optionsSelectedArr = ['0', '', '0'];

        // id가 'editor1'인 Obj정보의 input 객체를 새로 생성하여 삽입합니다.
        DEXT5.insertSelect(Obj, optionsTextArr, optionsValueArr, optionsSelectedArr, null, 'editor1');
        // work something ...
    }

</script>
