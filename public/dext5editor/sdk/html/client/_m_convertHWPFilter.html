DEXT5 Editor :: convertHWPFilter

string convertHWPFilter(strHtml, editor_id)

html 소스를 hwp(한글파일)포맷으로 변경하여 반환합니다.

return value

hwp(한글파일) 포맷으로 전환된 html소스

parameters

strHtml  변경할 html 소스를 의미합니다.
param  에디터 작성영역에서 추출할 태그의 ID를 의미합니다. ("" 입력시 전체, {extraID:"....."} 로 적용)

remarks

없음.
