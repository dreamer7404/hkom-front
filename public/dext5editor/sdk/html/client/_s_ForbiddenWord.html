DEXT5 Editor :: ForbiddenWord

XML : <forbidden_word>.....</forbidden_word>, Config : config.ForbiddenWord
XML : <forbidden_word mode="....."></forbidden_word>, Config : config.ForbiddenWordMode
XML : <forbidden_word width="....."></forbidden_word>, Config : config.ForbiddenWordWidth
XML : <forbidden_word height="....."></forbidden_word>, Config : config.ForbiddenWordHeight

Config 설정이 우선 적용됩니다.
금지어 목록을 설정합니다. 금지어 다이얼로그의 모드와 다이얼로그 창의 너비 높이를 설정합니다.

remarks

forbidden_word   콤마(,)로 구분하여 금지어를 입력합니다.
forbidden_word@mode   금지어 추출 모드 설정합니다.
                               (0입력 시 일반 다이얼로그, 다이얼로그 크기 고정, 금지어가 없어도 다이얼로그 출력)
                               (1입력 시 개선된 다이얼로그, 다이얼로그 크기 조절 가능, 금지어 없으면 alert출력, 다이얼로그 출력시 전체 Content도 출력)
forbidden_word@width   금지어 추출 다이얼로그의 가로길이를 설정합니다.
forbidden_word@height   금지어 추출 다이얼로그의 세로길이를 설정합니다.

sample code

에디터의 금지어를 "가나", "다라"로 설정합니다.
금지어 기능 사용시개선된 금지어 다이얼로그를 width:600px, height:600px로 출력하도록 설정합니다.

XML

.....
<setting>
    .....
    <forbidden_word mode="1", width="600px", height="600px">가나,다라</forbidden_word>
    .....
</setting>
.....
