DEXT5 Editor :: getMultiValue

String getMultiValue(editor_id, key)

해당 id의 에디터에서 saveCurrValueInMultiValue API로 저장된 MultiValue를 반환해줍니다.

return value

해당 에디터 id와 key로 저장된 MultiValue값

parameters

editor_id  에디터의 id를 의미합니다.
key  저장된 MultiValue에 해당하는 key값을 의미합니다.

remarks

saveCurrValueInMultiValue API로 에디터의 작성 내용을 MultiValue로 추가할 수 있습니다.

sample code

        var dext5 = DEXT5.getMultiValue('editor1', 'DEXT5');

    }
