DEXT5 Editor :: ImageEditorUse

XML : <image_editor_use>.....</image_editor_use>, Config : config.ImageEditorUse

Config 설정이 우선 적용됩니다.
미니 이미지 편집기 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 미니 이미지 편집기를 사용합니다.

sample code

미니 이미지 편집기를 사용하도록 설정합니다.

XML

.....
<license>
    <image_editor_use>1</image_editor_use>
    .....
</license>
.....

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
