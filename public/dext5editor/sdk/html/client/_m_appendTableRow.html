DEXT5 Editor :: appendTableRow

void appendTableRow(tableID, inHTML, rowIdx, upDown, editor_id)

에디터 작성영역에 해당 id를 가진 table이 있으면 행을 추가 합니다.

return value

없음.

parameters

tableID  에디터 body에 존재하는 table의 id를 의미합니다.
inHTML  추가하는 행 셀들의 내용값을 의미합니다. (구분자:\U000c)
rowIdx  추가할 위치의 index를 의미합니다.
upDown  rowIdx를 기준으로 추가할 위치를 의미합니다. (0:아래, 1:위)
editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.
