DEXT5 Editor :: Undo

XML : <undo mode="....."></undo>, Config : config.UndoMode
XML : <undo limit_length="....."></undo>, Config : config.UndoLimitLength
XML : <undo_count>.....</undo_count>, Config : config.UndoCount
XML : <light_mode>.....</light_mode>, Config : config.UseUndoLightMode
XML : <allow_delete_count>.....</allow_delete_count>, Config : config.AllowDeleteCount

Config 설정이 우선 적용됩니다.
Undo, Redo에 관련된 내용을 설정합니다.

remarks

mode  undo, redo의 모드를 설정합니다. (1로 설정 시 기존모드로 동작, 2로 설정 시 word와 유사방식으로 동작합니다.)
limit_length  set되는 데이터가 특정 길이 이상이면 undo, redo를 동작시키지 않도록 설정합니다. (0이상의 수로 설정 시 동작합니다.)
undo_count  글 작성중 설정값의 글자수 작성 시 마다 undo 지점으로 추가합니다. (글자 수를 입력합니다.)
light_mode  backSpace, delete, 엔터키 단순 입력 시에는 undo에 담지 않고,
               셀렉션한 후 해당키를 입력할때만 undo에 담도록 설정합니다.(1로 설정 시 적용됩니다.)
allow_delete_count  del, backSpace 키 입력 시 undo Count를 사용하도록 설정합니다. (1로 설정 시 적용됩니다.)

sample code

Undo, Redo에 관련된 내용을 설정합니다.

XML

.....
<setting>
    .....
    <undo mode="2" limit_length="500">
        <undo_count>30</undo_count>
        <light_mode>1</light_mode>
        <allow_delete_count>1</allow_delete_count>
    </undo>
    .....
</setting>
.....
