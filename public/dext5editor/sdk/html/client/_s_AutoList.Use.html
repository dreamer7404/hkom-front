DEXT5 Editor :: AutoList.Use

XML : <auto_list use="....."></auto_list>, Config : config.AutoList.Use

Config 설정이 우선 적용됩니다.
글 작성시 자동글머리 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 으로 설정 시 글 작성시 자동글머리 기능을 사용합니다.

sample code

자동글머리 기능을 사용합니다.

XML

.....
<setting>
    <auto_list use="1">1</auto_list>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
