[웹표준모드 전용]

DEXT5 Editor :: ImageContinueInsertMaintainValue

XML : <image_continue_insert_maintain_value>.....</image_continue_insert_maintain_value>
Config : config.ImageContinueInsertMaintainValue

Config 설정이 우선 적용됩니다.
웹표준모드에서 이미지 계속 삽입시 width,height,id를 제외한 나머지 값의 유지여부를 설정합니다.

remarks

기본값은 "0" 이며 사용 안함,
"1" 로 설정 시 웹표준모드에서 이미지 계속 삽입시 width,height,id를 제외한 나머지 값을 유지합니다.

sample code

웹표준모드에서 이미지 계속 삽입시 width,height,id를 제외한 나머지 값을 유지합니다. 

XML

.....
<setting>
    <image_continue_insert_maintain_value>1</image_continue_insert_maintain_value>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ImageContinueInsertMaintainValue = "1";

    </script>
