DEXT5 Editor :: UploadMethod

XML : <uploader_setting method="....."></uploader_setting>, Config : config.UploadMethod

Config 설정이 우선 적용됩니다.
이미지 업로드시 서버에 업로드 혹은 서버에 업로드 하지않고 이미지 태그 src에 base64로 추가할지 설정합니다.

remarks

기본값은 "upload" 이며 이미지 업로드 시 서버에 이미지 업로드 및 이미지 태그 src를 해당 업로드 된 경로로 지정하고,
"base64" 로 설정하는 경우, 이미지 업로드시 서버에 이미지를 업로드 하지 않고 이미지 태그 src를 해당 이미지의 base64로 지정합니다.

sample code

이미지 업로드시 서버에 이미지를 업로드 하지 않고 이미지 태그 src를 해당 이미지의 base64로 지정합니다.

XML

.....
<uploader_setting>
    .....
    <uploader_setting method="base64"></uploader_setting>
    .....
</uploader_setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">   
    <script type="text/javascript">

        DEXT5.config.UploadMethod = "base64";

        new Dext5editor("editor1");

    </script>       
</div>
