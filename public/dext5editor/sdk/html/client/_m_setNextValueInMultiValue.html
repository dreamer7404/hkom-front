DEXT5 Editor :: setNextValueInMultiValue

List setNextValueInMultiValue(editor_id, new_key)

에디터의 작성영역을 해당 key값으로 저장된 MultiValue의 값으로 변경합니다.

return value

해당 MultiValue를 포함한 List.

parameters

editor_id  에디터의 id를 의미합니다.
new_key  불러올 MultiValue의 key값을 의미합니다.

remarks

변경하는 것이기 때문에 저장을 안했을 시 기존에 작성된 내용은 전부 사라집니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Set(){
        // id가 editor1인 에디터의 작성영역을 "DEXT5"의 key로 저장된 MultiValue의 값으로 변경합니다.
        var value = DEXT5.setNextValueInMultiValue('editor1','DEXT5');
        alert(value);
        // work something ...
    }

</script>
