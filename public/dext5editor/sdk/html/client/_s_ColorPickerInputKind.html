DEXT5 Editor :: ColorPickerInputKind

XML : <color_picker_input_kind>.....</color_picker_input_kind>, Config : config.ColorPickerInputKind

Config 설정이 우선 적용됩니다.
Color Picker의 색상 입력 방식을 설정합니다.

remarks

기본값은 "0" 이고 HEX 값을 입력 방식으로 설정하고,
"1"로 설정 시 RGB 값을 입력 방식으로 설정합니다.

sample code

Color Picker의 색상 입력 방식을 RGB값으로 설정합니다. 

XML

.....
<setting>
    .....
    <color_picker_input_kind>1</color_picker_input_kind>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
