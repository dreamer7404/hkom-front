DEXT5 Editor :: ReturnEvent

XML : <return_event command_event="....."></return_event>, Config : config.ReturnEventCommand
XML : <return_event drag_event="....."></return_event>, Config : config.ReturnEventDrag
XML : <return_event focus_event="....."></return_event>, Config : config.ReturnEventFocus
XML : <return_event input_event="....."></return_event>, Config : config.ReturnEventInput
XML : <return_event keyboard_event="....."></return_event>, Config : config.ReturnEventKeyboard
XML : <return_event mouse_event="....."></return_event>, Config : config.ReturnEventMouse

Config 설정이 우선 적용됩니다.
에디터의 이벤트들의 활성화/비활성화 여부를 설정합니다.

remarks

아래 모든 설정값의 기본값은 "0" 이며 설정값이 "0"일 경우 해당 이벤트 발생하지 않습니다.
command_event  "1"로 설정시 dext_command_event를 사용 가능하도록 설정합니다.
drag_event  "1"로 설정시 dext_drag_event를 사용 가능하도록 설정합니다.
focus_event  "1"로 설정시 dext_focus_event 사용 가능하도록 설정합니다.
input_event  "1"로 설정시 dext_input_event를 사용 가능하도록 설정합니다.
keyboard_event  "1"로 설정시 dext_key_event를 사용 가능하도록 설정합니다.
mouse_event  "1"로 설정시 dext_mouse_event를 사용 가능하도록 설정합니다.

sample code

plugin 설치관련 내용을 설정합니다.

XML

.....
<setting>
    <return_event command_event="1" drag_event="1" focus_event="1" input_event="1" keyboard_event="1" mouse_event="1"></return_event>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ReturnEventCommand = "1";
        DEXT5.config.ReturnEventDrag = "1";
        DEXT5.config.ReturnEventFocus = "1";
        DEXT5.config.ReturnEventInput = "1";
        DEXT5.config.ReturnEventKeyboard = "1";
        DEXT5.config.ReturnEventMouse = "1";

        new Dext5editor("editor1");

    </script>       
</div>
