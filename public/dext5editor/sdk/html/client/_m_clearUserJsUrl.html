DEXT5 Editor :: clearUserJsUrl

void clearUserJsUrl(editor_id)

addUserJsUrl로 추가된 script를 모두 제거합니다.

return value

없음.

parameters

editor_id  적용할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        var url = 'http://www.dext5.com/dext5editor/js/test.js'
        DEXT5.addUserJsUrl(url, 'editor1');
        // id가 editor1인 에디터의 편집영역 iframe document의 head 영역에 addUserJsUrl로 추가된 script를 모두 제거합니다.
        DEXT5.clearUserJsUrl('editor1');
    }
        
</script>

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">
  
    <!-- 에디터 생성 -->
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>       

</div>
