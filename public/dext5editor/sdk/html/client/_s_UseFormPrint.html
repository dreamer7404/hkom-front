DEXT5 Editor :: UseFormPrint

XML : <use_form_print>.....</use_form_print>, Config : config.UseFormPrint

Config 설정이 우선 적용됩니다.
이미지 캡쳐 방식의 프린트 기능 사용여부를 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 이미지 캡쳐 방식의 프린트 기능을 사용합니다.
(설정시 UsePrintApp의 값도 "1"로 설정됩니다.)

sample code

이미지 캡쳐 방식의 프린트 기능을 사용합니다.

XML

.....
<license>
    <use_form_print>1</use_form_print>
    .....
</license>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.UseFormPrint = "1";

        new Dext5editor("editor1");

    </script>       
</div>
