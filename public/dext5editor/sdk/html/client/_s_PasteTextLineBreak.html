DEXT5 Editor :: PasteTextLineBreak

XML : <paste_text_line_break>.....</paste_text_line_break>, Config : config.PasteTextLineBreak

Config 설정이 우선 적용됩니다.
텍스트 붙여넣기시 <br> 대신 p 태그로 줄바꿈 되도록 설정합니다.

remarks

기본값은 "0" 이며 텍스트 붙여넣기시 줄바꿈을 <br>로 설정하며,
"1" 로 설정 시 텍스트를 붙여넣을 경우 줄바꿈을 <p><br><p>로 설정합니다.

sample code

텍스트를 붙여넣을 경우 줄바꿈을 <p><br><p>로 설정합니다.

XML

.....
<setting>
    <paste_text_line_break>1</paste_text_line_break>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.PasteTextLineBreak = "1";

        new Dext5editor("editor1");

    </script>       
</div>
