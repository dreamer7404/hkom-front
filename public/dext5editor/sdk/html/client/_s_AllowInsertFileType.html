DEXT5 Editor :: AllowInsertFileType

XML : <allow_insert_file_type>.....</allow_insert_file_type>
Config : config.AllowInsertFileType
XML : <allow_insert_file_type max_file_size="....."></allow_insert_file_type>
Config : config.MaxInsertFileSize

Config 설정이 우선 적용됩니다.
파일 삽입 시 허용할 확장자를 설정합니다.
파일 삽입 시 허용할 최대 파일 사이즈를 제한합니다.

remarks

allow_insert_file_type   확장자를 ","(콤마)로 구분하여 입력합니다.
max_file_size   용량을 바이트단위까지 입력합니다.

sample code

파일 삽입 시 허용할 확장자를 "docx,pptx"로 설정합니다.
파일 삽입 시 허용할 최대 파일 사이즈를 "10MB"로 설정합니다.

XML
.....
<uploader_setting>
    .....
    <allow_insert_file_type max_file_size="10MB">docx,pptx</allow_insert_file_type>
    .....
</uploader_setting>
.....
