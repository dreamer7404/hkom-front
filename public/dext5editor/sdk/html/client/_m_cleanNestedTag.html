DEXT5 Editor :: cleanNestedTag

void cleanNestedTag(confirmMessage, cleanHtmlOrNode, cleanType, cleanNestedCount, tagName, editor_id)

중복 검사할 html이나 node를 파라미터로 보내고, 중복 태그 검사가 필요한 경우 confirmMessage 를 띄운 후 수락 시 중복 검사를 진행합니다.

return value

targetNode에서 html 태그가 정리된 node.

parameters

confirmMessage  confirm에 출력할 메시지를 설정합니다.
cleanHtmlOrNode  중복 검사할 html or node를 설정합니다.
cleanType  검사할 영역을 설정합니다. (1 : 중첩항목만, 2 : 전체영역)
cleanNestedCount  중복 태그 개수를 설정합니다. 해당 개수 이상일 경우 중복 검사를 진행합니다.
tagName  중복 검사를 할 태그를 설정합니다.
editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.
