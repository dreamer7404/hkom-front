DEXT5 Editor :: UnderlineAndStrikeThroughMode

XML : <underline_and_strike_through_mode>.....</underline_and_strike_through_mode>,
Config : config.UnderlineAndStrikeThroughMode

Config 설정이 우선 적용됩니다.
언더라인, 취소선을 span의 style로 적용 또는 u, s태그로 적용을 선택하여 설정합니다.

remarks

"1"으로 설정 시 언더라인, 취소선을 span의 style로 적용하고, "2"로 설정 시 u, s태그로 적용합니다.
기본값은 "2" 입니다.

sample code

언더라인, 취소선을 span의 style로 적용하도록 설정합니다.

XML

.....
<setting>
    .....
    <underline_and_strike_through_mode>1</underline_and_strike_through_mode>
    .....
</setting>
.....
