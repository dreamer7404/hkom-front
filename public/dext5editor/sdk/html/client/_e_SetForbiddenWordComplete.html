DEXT5 Editor :: Event.SetForbiddenWordComplete

void SetForbiddenWordComplete(editorId, paramObject)

금지어 추출 다이얼로그가 적용, 취소, 닫기 등으로 완료 시 발생합니다.

return value

없음.

parameters

editorId  이벤트가 발생한 에디터의 id입니다.
paramObject  금지어추출 적용 후 정보를 담은 객체 입니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 setting 설정 forbidden_word의 mode 속성값이 1인 경우에 이벤트가 발생합니다.
paramObject는 아래의 정보를 담고 있습니다.
 - paramObject.callbackStatus = 적용한 동작 (cancle || apply)
 - paramObject.callbackType = 기능을 호출한 방식 (API || ICON)
 - paramObject.permitWord = 금지어 추출에 걸린 항목들 중 적용 후 남은 금지어 리스트(중복O)
 - paramObject.permitWordToDeduplication = 금지어 추출에 걸린 항목들 중 적용 후 남은 금지어 리스트(중복X)
