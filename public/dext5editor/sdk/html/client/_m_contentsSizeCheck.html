DEXT5 Editor :: contentsSizeCheck

void contentsSizeCheck(includeExternalUrl, maxContentsSize, callBackFunc, editor_id)

maxContentsSize 파라미터에 입력한 값과 에디터 내의 Contents 전체 파일의 용량을 비교합니다.

return value

없음.

parameters

includeExternalUrl  외부에서 추가한 content의 size를 포함할지 여부를 의미합니다.(0 : 추가안함, 1 : 추가함)
maxContentsSize  임의로 비교할 에디터의 최대 contents 사이즈를 의미합니다.
callBackFunc  callBack 함수를 의미합니다. (첫번째 파라미터로 boolean값을 가져옵니다. (maxContentsSize < totalSize))
editor_id  적용할 에디터의 id를 의미합니다.

remarks

없음.
