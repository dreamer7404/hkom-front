DEXT5 Editor :: RemoveTdStyleInPastePpt

XML : <remove_td_style_in_paste_ppt>.....</remove_td_style_in_paste_ppt>
Config : config.RemoveTdStyleInPastePpt

Config 설정이 우선 적용됩니다.
PPT에서 테이블 복사후 에디터에 붙여넣기 시 TD에 적용되는 Default Style을 제거하도록 설정합니다.
(Default Style : font-size: 18.0pt; font-family: Arial;)

remarks

기본값은 "0"으로 사용 안함,
"1"로 설정 시 PPT에서 테이블 복사후 에디터에 붙여넣기 시 TD에 적용되는 Default Style을 제거합니다.

sample code

PPT에서 테이블 복사후 에디터에 붙여넣기 시 TD에 적용되는 Default Style을 제거합니다.

XML

.....
<setting>
    <remove_td_style_in_paste_ppt>0</remove_td_style_in_paste_ppt>
    .....
</setting>
.....
