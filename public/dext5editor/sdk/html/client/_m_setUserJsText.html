DEXT5 Editor :: setUserJsText

void setUserJsText(jsText, editor_id)

에디터 편집영역 document의 head 부분에 user의 js 스크립트문을 추가합니다.

return value

없음.

parameters

jsText  js스크립트문을 의미합니다.
editor_id  에디터의 id를 의미합니다.

remarks

추가 후 즉시 동작됩니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5Set(){
        // 테스트용 js 스크립트문 설정
        var jsscr = "alert('DEXT5 Editor')";
        // id가 editor1인 에디터 편집영역 document의 head 부분에 user의 js 스크립트문을 추가합니다.
        DEXT5.setUserJsText(jsscr, 'editor1');
        // work something ...
    }

</script>	

<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px;"> 
 
    <!-- 에디터 생성 -->  
    <script type="text/javascript">
        new Dext5editor('editor1');
    </script>

</div>
