DEXT5 Editor :: convertMMToPxUnit

string convertMMToPxUnit(strHtml)

태그 안의 width와 height 중 mm단위가 존재하면 px단위로 변경하여 반환합니다.

return value

태그 안의 width와 height 중 mm단위를 px단위로 변경한 html소스

parameters

strHtml  변경할 html 소스를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext_editor_loaded_event() {
        // 테스트용 html을 생성합니다.
        var html = '';
        html += '<head>';
        html += '<title>DEXT5 Editor</title>';
        html += '<meta content="text/html" charset="utf-8" http-equiv="Content-Type">'; 
        html += '</head>';
        html += '<body>';
        html += '<iframe style="background: gray;" width="500mm" height="300mm"></iframe>';
        html += '</body>';

        // html소스의 style 부분중 크기단위를 모두 px로 변경한 값을 반환받습니다.
        var text = DEXT5.convertMMToPxUnit(html);
        alert(text);
    }
        
</script>
