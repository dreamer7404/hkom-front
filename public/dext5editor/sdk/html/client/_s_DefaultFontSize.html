DEXT5 Editor :: DefaultFontSize

XML : <font_size>.....</font_size>, Config : config.DefaultFontSize
XML : <font_size inc_dec_gap="....."></font_size>, Config : config.FontSizeIncDecGap

Config 설정이 우선 적용됩니다.
에디터 로드 시 기본으로 사용할 폰트사이즈를 설정합니다.
tool bar의 폰트사이즈 콤보박스의 크기조정 화살표의 증감 정도를 설정합니다.

remarks

font_size  에디터 로드 시 사용할 기본 폰트사이즈를 입력합니다.
font_size@inc_dec_gap  폰트사이즈 콤보박스의 크기조정 화살표의 증감 정도를 입력합니다.

sample code

에디터 로드 시 사용할 기본 폰트사이즈를 16pt로 설정합니다.
tool bar의 폰트사이즈 콤보박스의 크기조정 화살표의 증감 정도를 2로 설정합니다.

XML

.....
<setting>
    <font_size inc_dec_gap="2">16pt</font_size>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
