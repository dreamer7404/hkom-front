DEXT5 Editor :: getMultiValueLength

String getMultiValueLength(editor_id)

해당 id의 에디터에서 saveCurrValueInMultiValue API로 저장된 MultiValue의 수를 반환해줍니다.

return value

해당 에디터 id에 저장된 MultiValue의 개수

parameters

editor_id  에디터의 id를 의미합니다.

remarks

saveCurrValueInMultiValue API로 에디터의 작성 내용을 MultiValue로 추가할 수 있습니다.
