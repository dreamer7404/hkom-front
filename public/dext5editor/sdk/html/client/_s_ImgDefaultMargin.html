DEXT5 Editor :: ImgDefaultMargin

XML : <img_default_margin margin_bottom="....."></img_default_margin>
Config : config.ImgDefaultMarginBottom
XML : <img_default_margin margin_left="....."></img_default_margin>
Config : config.ImgDefaultMarginLeft
XML : <img_default_margin margin_right="....."></img_default_margin>
Config : config.ImgDefaultMarginRight
XML : <img_default_margin margin_top="....."></img_default_margin>
Config : config.ImgDefaultMarginTop

Config 설정이 우선 적용됩니다.
이미지 다이얼로그의 상, 하, 좌, 우 기본 여백을 설정합니다.
이미지 삽입시가 아닌 다이얼로그 팝업시 기본적으로 입력되어있는 값을 설정합니다.

remarks

margin_bottom  이미지 다이얼로그에 입력될 기본 하여백을 설정합니다.
margin_left  이미지 다이얼로그에 입력될 기본 좌여백을 설정합니다.
margin_right  이미지 다이얼로그에 입력될 기본 우여백을 설정합니다.
margin_top  이미지 다이얼로그에 입력될 기본 상여백을 설정합니다.

sample code

이미지 다이얼로그의 상, 하, 좌, 우 기본 여백을 6으로 설정합니다.

XML

.....
<setting>
    .....
    <img_default_margin margin_left="6" margin_right="6" margin_top="6" margin_bottom="6"></img_default_margin>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">
