DEXT5 Editor :: AllowImgSize

XML : <allow_img_size>.....</allow_img_size>, Config : config.AllowImgSize

Config 설정이 우선 적용됩니다.
다이얼로그에서 이미지 삽입시 가로 세로 크기설정의 사용여부를 설정합니다.

remarks

기본값은 "1" 이며 다이얼로그에서 이미지 삽입시 가로 세로 크기설정을 사용,
"0" 으로 설정 시 사용하지 않습니다.

sample code

다이얼로그에서 이미지 삽입시 가로 세로 크기설정을 사용하지 않습니다.

XML

.....
<setting>
    <allow_img_size>0</allow_img_size>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.AllowImgSize = "0";

        new Dext5editor("editor1");

    </script>       
</div>
