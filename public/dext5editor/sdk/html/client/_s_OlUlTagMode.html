DEXT5 Editor :: OlUlTagMode

XML : <ol_ul_tag_mode>.....</ol_ul_tag_mode>, Config : config.OlUlTagMode

Config 설정이 우선 적용됩니다.
set API 사용시 개선된 글머리 기호, 번호의 사용여부를 설정합니다.

remarks

기본값은 "1" 이며 기존 글머리 기호, 번호를 사용,
"2" 로 설정 시 개선된 글머리 기호, 번호를 사용합니다.
