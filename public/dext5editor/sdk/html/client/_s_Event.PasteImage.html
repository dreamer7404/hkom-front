DEXT5 Editor :: Event.PasteImage

config.Event.PasteImage

callback 이벤트를 설정합니다.

remarks

에디터의 설정파일(dext5editor/config/dext_editor.xml)의 event_for_paste_image 의 설정값이 1인 경우에 이벤트가 발생합니다.
해당 속성값은 script에서 DEXT5.config.EventForPasteImage = "1"; 로도 설정이 가능합니다.

dext_paste_image_event를 설정합니다.
이미지를 붙여넣기 한 후 (완료 후) 이벤트가 발생합니다.
config.Event에 Object로 Event들을 모아서 설정이 가능합니다.

sample code

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        // 이벤트발생시 호출할 함수를 설정합니다.
        var eventFunc = function(eventParam1, eventParam2, eventParam3){
            alert("Event!");
            // work something ...
        }

        // PasteImage 이벤트를 사용하도록 설정합니다. (config 설정파일에서도 가능합니다.)
        DEXT5.config.EventForPasteImage = "1";

        // 이미지를 붙여넣기 한 후 (완료 후) 이벤트가 발생합니다. (Object로 모아서 설정이 가능합니다.)
        DEXT5.config.Event = {
            PasteImage : eventFunc,
            // Add Another Event Config ...
        }

        /* 다음과 같이 설정도 가능합니다.
        DEXT5.config.Event.PasteImage = eventFunc;
        */

        new Dext5editor("editor1");
    </script>       
</div>
