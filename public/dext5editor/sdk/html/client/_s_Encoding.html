DEXT5 Editor :: Encoding

XML : <encoding>.....</encoding>, Config : config.Encoding

Config 설정이 우선 적용됩니다.
에디터 디자인 모드에 사용할 문자 인코딩(character encoding) 형태를 설정합니다.

remarks

다국어 지원을 위해서는 "utf-8" 사용을 권장하며, DEXT5 Editor의 기본값도 "utf-8" 입니다.
사이트의 구성에 맞게 "euc-kr" 등을 값으로 사용할 수 있습니다.

sample code

에디터 디자인 모드에 사용할 문자 인코딩 형태를 euc-kr로 설정합니다.

XML

.....
<setting>
    .....
    <encoding>euc-kr</encoding>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
 
<!-- ..... 생략 ..... -->
 
<div style="width:900px;height:550px">    
    <script type="text/javascript">
        
        DEXT5.config.Encoding = "euc-kr";

        new Dext5editor("editor1");

    </script>       
</div>
