DEXT5 Editor :: ApplyStyleEmptyTag

XML : <apply_style_empty_tag>.....</apply_style_empty_tag>, Config : config.ApplyStyleEmptyTag

Config 설정이 우선 적용됩니다.
내용이 없는 빈 영역에도 서식 또는 스타일을 적용하도록 설정합니다.

remarks

기본값은 "0" 이며 사용하지 않음,
"1" 로 설정 시 내용이 없는 빈 영역에도 서식 또는 스타일을 적용합니다.

sample code

내용이 없는 빈 영역에도 서식 또는 스타일을 적용합니다.

XML

.....
<setting>
    <apply_style_empty_tag>1</apply_style_empty_tag>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.ApplyStyleEmptyTag = "1";
