DEXT5 Editor :: isPluginInstalled

Boolean isPluginInstalled(editor_id)

ieplugin이 설치되어있는지의 여부를 확인합니다.

return value

ieplugin 설치 여부

parameters

editor_id  에디터의 id를 의미합니다.

remarks

plugin모드가 아닐 때는 false를 반환합니다.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function dext5plugin(){
        // ieplugin이 설치되어있는지의 여부를 확인합니다.
        var dext5 = DEXT5.isPluginInstalled('editor1');
        alert(dext5)
        // work something ...
    }

</script>
