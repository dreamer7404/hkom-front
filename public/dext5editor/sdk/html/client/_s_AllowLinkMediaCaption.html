DEXT5 Editor :: AllowLinkMediaCaption

XML : <allow_link_media_caption>.....</allow_link_media_caption>, Config : config.AllowLinkMediaCaption

Config 설정이 우선 적용됩니다.
외부동영상 공유 다이얼로그에서 설명작성(caption) 부분의 활성화 여부를 설정합니다.

remarks

기본값은 "0" 이며 활성화하지 않음,
"1" 로 설정 시 외부동영상 공유 다이얼로그에서 설명작성(caption) 부분을 활성화합니다.

sample code

외부동영상 공유 다이얼로그에서 설명작성(caption) 부분을 활성화합니다.

XML

.....
<setting>
    <allow_link_media_caption>1</allow_link_media_caption>
    .....
</setting>
.....

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>
