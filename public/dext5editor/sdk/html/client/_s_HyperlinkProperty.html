DEXT5 Editor :: HyperlinkProperty

XML : <category>.....</category>, Config : config.HyperLinkCategoryList
XML : <click_ctrl_hyperlink>.....</click_ctrl_hyperlink>, Config : config.ClickCtrlHyperlink
XML : <protocol>.....</protocol>, Config : config.HyperLinkProtocolList
XML : <target>.....</target>, Config : config.HyperLinkTargetList
XML : <target basic_list="....."></target>, Config : config.HyperlLinkTargetBasicList

Config 설정이 우선 적용됩니다.
하이퍼링크 삽입 다이얼로그 및 하이퍼링크 속성을 설정합니다.

remarks

category  하이퍼링크 category를 콤마(,)로 구분하여 설정합니다. (hyperlink,bookmark,image 등)
click_ctrl_hyperlink  기본값은 "0" 이고 사용 안함, "1"로 설정 시 디자인 모드에서 하이퍼링크를 Ctrl + 클릭하면 바로 이동하도록 설정합니다.
protocol  하이퍼링크 클릭시 프로토콜을 콤마(,)로 구분하여 설정합니다.(http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news: 등)
target  하이퍼링크 다이얼로그의 대상프레임 항목 콤보박스에 임의의 value를 콤마(,)로 구분하여 추가합니다.
          (해당 내용으로 설정시 생성한 하이퍼링크의 target에 해당 value가 적용됩니다.)
target@basic_list  하이퍼링크 다이얼로그의 대상프레임 항목 콤보박스에 기본항목을 콤마(,)로 구분하여 설정합니다.

sample code

하이퍼링크 삽입 다이얼로그 및 하이퍼링크 속성을 설정합니다.

XML

.....
<setting>
    .....
    <hyperlink_property>
        <category>hyperlink,bookmark,image</category>
        <click_ctrl_hyperlink>1</click_ctrl_hyperlink>
        <protocol>http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news:</protocol>
        <target basic_list="blank,self,top,parent">_blank</target>
    </hyperlink_property>
    .....
</setting>
.....

Config

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
    
<!-- ..... 생략 ..... -->
    
<div style="width:900px;height:550px">    
    <script type="text/javascript">

        DEXT5.config.HyperLinkCategoryList = "hyperlink,bookmark,image";
        DEXT5.config.ClickCtrlHyperlink = "1";
        DEXT5.config.HyperLinkProtocolList = "http://,https://,file:///,ftp://,gopher://,telnet://,wais://,mailto:,news:";
        DEXT5.config.HyperLinkTargetList = "_blank";
        DEXT5.config.HyperlLinkTargetBasicList = "blank,self,top,parent";

        new Dext5editor("editor1");

    </script>       
</div>
