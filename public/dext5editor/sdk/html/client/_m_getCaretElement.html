DEXT5 Editor :: getCaretElement

Element getCaretElement(editor_id)

에디터의 작성영역 현재 커서위치에 해당하는 element를 반환해줍니다.

return value

에디터 커서위치의 element

parameters

editor_id  API를 적용할 에디터의 id를 의미합니다.

remarks

없음.

sample code

<!-- ..... 생략 ..... -->

<script type="text/javascript" src="dext5editor/js/dext5editor.js"></script>	
<script type="text/javascript">

    function caretElement(){
        // id가 editor1인 에디터의 작성영역 현재 커서위치에 해당하는 element를 반환받습니다.
        var caretElement = DEXT5.getCaretElement('editor1');
        alert(caretElement.outerHTML);
    }

</script>
